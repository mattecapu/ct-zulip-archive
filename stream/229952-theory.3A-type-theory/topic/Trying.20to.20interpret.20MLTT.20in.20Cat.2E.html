Topic: Trying to interpret MLTT in Cat.

finegeometer (Aug 16 2023 at 15:54):

Inspired by the search for a directed version of homotopy type theory, I've been trying to interpret ordinary MLTT type theory in the 2-category of categories. It seems to almost work.

Here's what I have so far:

  • To each context Γ\GammaΓ, assign a 1-category ⟦Γ⟧\llbracket\Gamma\rrbracket[[Γ]].
  • To each type Γ⊢X type\Gamma \vdash X \textsf{ type}Γ⊢X type, assign a pseudofunctor ⟦X⟧:⟦Γ⟧→Cat\llbracket X\rrbracket : \llbracket\Gamma\rrbracket \to \mathbf{Cat}[[X]]:[[Γ]]→Cat.
  • To each value Γ⊢x:X\Gamma \vdash x : XΓ⊢x:X, assign a lax natural transformation ⟦x⟧:(_↦1)⇒lax⟦X⟧\llbracket x\rrbracket : (\_ \mapsto \mathbf{1}) \Rightarrow_\mathrm{lax} \llbracket X\rrbracket[[x]]:(_↦1)⇒lax​[[X]].
  • For contexts, ⟦⋅⟧\llbracket\cdot\rrbracket[[⋅]] is the terminal category, and ⟦Γ,x:X⟧=∫⟦X⟧\llbracket\Gamma, x : X\rrbracket = \int\llbracket X\rrbracket[[Γ,x:X]]=∫[[X]]; the result of applying the Grothendieck construction to ⟦X⟧\llbracket X\rrbracket[[X]].

This all seems to work. But I'm having trouble with pi-types. Letting ppp be the canonical functor ∫⟦X⟧→⟦Γ⟧\int \llbracket X\rrbracket \to \llbracket\Gamma\rrbracket∫[[X]]→[[Γ]], I need a right 2-adjoint to p∗:[⟦Γ⟧,Cat]→[∫⟦X⟧,Cat]p^* : [\llbracket\Gamma\rrbracket, \mathbf{Cat}] \to [\int \llbracket X\rrbracket, \mathbf{Cat}]p∗:[[[Γ]],Cat]→[∫[[X]],Cat], where those functor categories are categories of pseudofunctors and lax natural transformations. But I'm not sure this adjoint actually exists.

I think I've found references that say I'd be fine if I were working with pseudonatural transformations instead. But if I do that, I have problems with the variable rule instead.

This is where I'm stuck. I don't know if it's just that my inexperience with 2-categories has caught up to me, or if the whole idea is fundamentally flawed.
