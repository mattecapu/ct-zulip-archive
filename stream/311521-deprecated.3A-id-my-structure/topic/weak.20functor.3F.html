Topic: weak functor?

Joshua Meyers (Jan 07 2022 at 09:36):

Consider a functor U:D→CU:D\to CU:D→C. If the comma category d/Ud/Ud/U has an initial object for each d∈Dd\in Dd∈D, then we can construct a left adjoint F:C→DF:C\to DF:C→D by sending ccc to the object d′d'd′, where (d′,f)(d',f)(d′,f) is initial in d/Ud/Ud/U . If d/Ud/Ud/U merely has a weakly initial object for each d∈Dd\in Dd∈D, then we can construct some sort of "weak functor" F:C→DF:C\to DF:C→D by sending ccc to the family of objects d′∈Dd'\in Dd′∈D such that (d′,f)(d',f)(d′,f) is weakly initial in d/Ud/Ud/U for some f:d→d′f:d\to d'f:d→d′. Anyone know a name for this sort of "weak functor"?
