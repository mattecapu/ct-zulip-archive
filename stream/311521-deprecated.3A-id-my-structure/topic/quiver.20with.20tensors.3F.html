Topic: quiver with tensors?

Samuel Gélineau (Nov 16 2022 at 18:49):

Is there a special name for the following structure? It's useful as a generator for a monoidal category, but it is not itself a category.

  • a set T of objects {A, B, ...}
  • a set of morphisms, each of which has an input type and an output type. A type is not an object, but a list of objects, written as a tensor A ⊗ B ⊗ C or as 1 for the empty list.

This is not a quiver, because T does not include 1 nor A ⊗ B.

For example, the following instance of that structure represents what sigfpe calls the "four essential components" of a tangle:

  • a singleton set of objects, where the single object is called "Wire"
  • the set of morphisms {over, under, cup, cap}, whose input and output types are as follows:
over : Wire ⊗ Wire -> Wire ⊗ Wire
under : Wire ⊗ Wire -> Wire ⊗ Wire
cup : Wire ⊗ Wire -> 1
cap : 1 -> Wire ⊗ Wire

A tangle, as illustrated later in the linked blog post, can then be defined as the free monoidal category generated by those essential components.
