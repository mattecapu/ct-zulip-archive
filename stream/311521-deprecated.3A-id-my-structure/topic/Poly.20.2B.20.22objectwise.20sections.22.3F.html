Topic: Poly + "objectwise sections"?

Bruno Gavranović (Feb 11 2023 at 17:04):

So I've got a category that is just like the category Poly\mathbf{Poly}Poly, but with more stuff (I'll call it Poly+\mathbf{Poly}^+Poly+). An object in Poly\mathbf{Poly}Poly is a pair (A,A′)(A, A')(A,A′), where A:SetA : \mathbf{Set}A:Set and A′:A→SetA' : A \to \mathbf{Set}A′:A→Set, but an object in Poly+\mathbf{Poly^+}Poly+ is a triple (A,A′,s)(A, A', s)(A,A′,s), where s:(a:A)→A′(a)s : (a : A) \to A'(a)s:(a:A)→A′(a), something like a section.

A morphism (A,A′)→(B,B′)(A, A') \to (B, B')(A,A′)→(B,B′) in Poly\mathbf{Poly}Poly is a pair (f,f♯)(f, f^\sharp)(f,f♯) where f:A→Bf : A \to Bf:A→B, and f♯:(a:A)→B′(f(a))→A′(a)f^\sharp: (a : A) \to B'(f(a)) \to A'(a)f♯:(a:A)→B′(f(a))→A′(a).
A morphism (A,A′,s)→(B,B′,r)(A, A', s) \to (B, B', r)(A,A′,s)→(B,B′,r) is the pair above + the condition that ∀a:A.f♯(a,r(f(a)))=s(a)\forall a : A . f^\sharp (a, r (f(a))) = s(a)∀a:A.f♯(a,r(f(a)))=s(a).

Is there some known categorical semantics of this?

Bruno Gavranović (Feb 11 2023 at 17:13):

If we restrict our dependent type theory to the one with only erased types, then this recovers something like a "dependent twisted arrow category".
If we then additionally require this to be non-dependent (but still erased), then this recovers a bona-fide twisted arrow category.
That is, if A′A'A′ and B′B'B′ as functions are constant (I'll call the objects they're constant on A′A'A′ and B′B'B′, overloading notation), then the sections from above turn into simple morphisms s:A→A′s : A \to A's:A→A′ and r:B→B′r : B \to B'r:B→B′.
The forward map stays the same f:A→Bf : A \to Bf:A→B, but the backward map f♯:A×B′→A′f^\sharp : A \times B' \to A'f♯:A×B′→A′ is erased in AAA, meaning it's defined by f♯:B′→A′f^\sharp : B' \to A'f♯:B′→A′. This is exactly a morphism in tw(C)tw(\mathcal{C})tw(C).
