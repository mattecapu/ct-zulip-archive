If E\mathcal{E}E is a category with finite limits then there is a bicategory of "split spans". A 1-cell in the bicategory of split spans from AAA to BBB is an object XXX equipped with f:X→A,g:X→Bf : X\to A, g : X\to Bf:X→A,g:X→B, h:A→Xh : A\to Xh:A→X with fh=1Afh=1_Afh=1A​.

Unit and Composition extends the unit and composition of spans. A 2-cell is a map of spans respecting the splitting.

Anybody see this before?

Similarly there should be a pseudo double category of E\mathcal{E}E objects, E\mathcal{E}E morphisms and split spans.

Intuitively if spans are generalized relations then these things are generalized total relations, they come with a function witnessing that the relation is total.

Here's why I think this is an interesting notion. Consider the bicategory of sets and spans. Let C\mathcal{C}C be a category and let F:C→SpanF : \mathcal{C}\to\mathbf{Span}F:C→Span be a lax functor. So FFF associates to each ccc a set F(c)F(c)F(c) and to each f:c→df : c\to df:c→d a family of sets F(f)x,yF(f)_{x,y}F(f)x,y​ for x∈F(c),y∈F(d)x\in F(c), y\in F(d)x∈F(c),y∈F(d). These relations are reflexive, so F(1c)(x,x)F(1_c)(x,x)F(1c​)(x,x) is true for every xxx, and transitive, so F(f)(x,y)∧F(g)(y,z)  ⟹  F(gf)(x,z)F(f)(x,y) \land F(g)(y,z)\implies F(gf)(x,z)F(f)(x,y)∧F(g)(y,z)⟹F(gf)(x,z).

If f:c→df : c\to df:c→d, then to prove a relation F(f)F(f)F(f) is "total valued" is to give an element of ∏x:F(c)∑y:F(d)F(f)(x,y)\prod_{x: F(c)}\sum_{y: F(d)}F(f)(x,y)∏x:F(c)​∑y:F(d)​F(f)(x,y).
If we want to prove this for all fff, there are natural coherence conditions that should be respected associated to the identity and composition of the category, and this entails that we lift the functor FFF to the bicategory of split spans.

For example, there is a functor F:Catop→SpanF : \mathbf{Cat}^{\rm op}\to \mathbf{Span}F:Catop→Span sending each CCC to Ob(C)\mathbf{Ob}(C)Ob(C) and R:D→CR : D\to CR:D→C to F(R)(c,d):={η:c→R(d)∣η is a universal arrow}F(R)(c,d):= \left\{ \eta : c\to R(d)\mid \eta \textrm{ is a universal arrow} \right\}F(R)(c,d):={η:c→R(d)∣η is a universal arrow}. RRR has a left adjoint iff FFF is a total-valued relation, i.e., if there is an element of ∏c∑dF(R)(c,d)\prod_{c}\sum_dF(R)(c,d)∏c​∑d​F(R)(c,d). If we have a diagram in Catop\mathbf{Cat}^{\rm op}Catop, then we can lift it to the bicategory of split spans iff it is possible to choose a left adjoint for each functor in the diagram in a way that is compatible with composition on-the-nose.

As another example, let p:E→Bp: \mathbb{E}\to \mathbb{B}p:E→B be a Grothendieck fibration, and let p−1:Bop→Spanp^{-1} : \mathbb{B}^{\rm op}\to \mathbf{Span}p−1:Bop→Span be the lax functor which sends bbb to Ob(p−1(b))\mathbf{Ob}(p^{-1}(b))Ob(p−1(b)) and for f:c→df : c\to df:c→d,x∈p−1(c),y∈p−1(d)x\in p^{-1}(c), y\in p^{-1}(d)x∈p−1(c),y∈p−1(d), p−1(f)(x,y)p^{-1}(f)(x,y)p−1(f)(x,y) is the set of Cartesian arrows from xxx to yyy over fff.

Giving a lift of this lax functor to the bicategory of split spans is equivalent to choosing a splitting for the fibration.
