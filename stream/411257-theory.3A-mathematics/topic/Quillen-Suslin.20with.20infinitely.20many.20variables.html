Tobias Fritz (Nov 06 2023 at 13:41):

Jean-Baptiste Vienney said:

(Maybe the theorem is true as an easy consequence of Quillen-Suslin in some way like this.)

Yes, I think this is what happens. To see this, for a fg projective module PPP, consider the composite

Rn≅P⊕Q→P→P⊕Q≅Rn,R^n \cong P \oplus Q \to P \to P \oplus Q \cong R^n ,Rn≅P⊕Q→P→P⊕Q≅Rn,

where all maps are the obvious ones. Then this map can be represented by a matrix e∈Mn×n(R)e \in M_{n \times n}(R)e∈Mn×n​(R) with e2=ee^2 = ee2=e. The module is free if and only if there is an isomorphism P≅RmP \cong R^mP≅Rm for some mmm, and using this we can also represent the projection P⊕Q→PP \oplus Q \to PP⊕Q→P as a matrix p∈Mn×m(R)p \in M_{n \times m}(R)p∈Mn×m​(R), and the inclusion P→P⊕QP \to P \oplus QP→P⊕Q as a matrix i∈Mm×n(R)i \in M_{m \times n}(R)i∈Mm×n​(R), such that e=ipe = ipe=ip and pi=1pi = 1pi=1. Like this, I believe that we get the following:

Theorem: A ring RRR has the property that every fg projective module is free if and only if every idempotent matrix with entries from RRR splits.

Now the punchline: over R=k[x1,x2,… ]R = k[x_1,x_2,\dots]R=k[x1​,x2​,…], every matrix can only contain finitely many variables! Like this, the problem is reduced to the Quillen-Suslin theorem, and it should indeed be true that every fg projective module is free. I'm saying "should" because I haven't carefully worked out the proof of the above theorem. But assuming that it's true, we can think of this geometrically as saying that every algebraic vector bundle in infinite dimensions can vary in only finitely many of them, and it's trivially trivial in the other directions.

Jens Hemelaer (Nov 06 2023 at 17:49):

In general, if P(R)\mathbf{P}(R)P(R) is the monoid of finitely generated projective RRR-modules, up to isomorphism,
then P\mathbf{P}P preserves filtered colimits, see here, 2.1.6.
In our situation, k[xi,i∈I]k[x_i, i \in I]k[xi​,i∈I] is the filtered colimit of the k[xi,i∈J]k[x_i, i \in J]k[xi​,i∈J] for JJJ going over the finite subsets of III.
And P(k[xi,i∈J])\mathbf{P}(k[x_i, i \in J])P(k[xi​,i∈J]) is always the monoid (N,+)(\mathbb{N},+)(N,+), so it is still this monoid of natural numbers in the limit.
