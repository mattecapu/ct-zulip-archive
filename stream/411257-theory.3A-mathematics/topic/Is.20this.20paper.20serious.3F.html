Here I think is one way to produce topological rings with nontrivial fundamental groups. Please let me know if you see any problems. It proceeds through the following steps.

  • The functor B:Mon→CatB: \mathsf{Mon} \to \mathsf{Cat}B:Mon→Cat, sending a monoid MMM to the obvious 1-object category BMBMBM, is product-preserving. Being product-preserving, it sends abelian group objects in Mon\mathsf{Mon}Mon (which are the same as monoid objects in (Ab,×)(\mathsf{Ab}, \times)(Ab,×), which are the same as abelian groups) to abelian group objects in Cat\mathsf{Cat}Cat. So for example, A=B(Z/2)A = B(\mathbb{Z}/2)A=B(Z/2) is an abelian group object in Cat\mathsf{Cat}Cat.

  • Cat\mathsf{Cat}Cat is cartesian closed. Given an abelian group object AAA in a cartesian closed category, its internal endo-hom object [A,A][A, A][A,A] carries a ring object structure, with addition being defined "internally pointwise", and multiplication given by internal composition. (For A=B(Z/2)A = B(\mathbb{Z}/2)A=B(Z/2), the underlying category of [A,A][A, A][A,A] is a finite category; this will be needed later.)

  • The nerve functor N:Cat→SetΔopN: \mathsf{Cat} \to \mathsf{Set}^{\Delta^{op}}N:Cat→SetΔop and the geometric realization functor R:SetΔop→TopR: \mathsf{Set}^{\Delta^{op}} \to \mathsf{Top}R:SetΔop→Top are both product-preserving. Let's put a pin through that; I'll come back to this technical point in the next bullet point. In any case, the functor RN:Cat→TopRN: \mathsf{Cat} \to \mathsf{Top}RN:Cat→Top, being product-preserving, takes ring objects in Cat\mathsf{Cat}Cat to ring objects in Top\mathsf{Top}Top, hence CGWH\mathrm{CGWH}CGWH-rings under one choice of notion of "convenient space".
