The definition on that nLab page leaves me a bit confused; I prefer your approach: an endoprofunctor $$H : C^{op}\times C \to Set$ corresponds to a left adjoint H^:[Cop,Set]→[Cop,Set]\hat H : [C^{op},Set] \to [C^{op},Set]H^:[Cop,Set]→[Cop,Set], and from the adjunction (H^⊣R)(\hat H\dashv R)(H^⊣R) one gets a monad on [Cop,Set][C^{op},Set][Cop,Set]; an algebra for HHH is an algebra for the monad RH^R\hat HRH^. The problem I have with this definition is that _every_ profunctor, not only an endoprofunctor, generates such a monad, whereas an "algebra" is usually defined as an object on which a monoid object is acting, being as an action map M⊗X→XM\otimes X \to XM⊗X→X or, by transposition, as a map of monoids M→End(X)M \to End(X)M→End(X).
