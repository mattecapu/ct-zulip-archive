Another example: We can take the reader monad TX=XATX = X^ATX=XA. This doesn't work in Set because Set satisfies AC, but it could work in another topos. For instance, we could take TX=XΔ1TX = X^{\Delta^1}TX=XΔ1 on simplicial sets.
