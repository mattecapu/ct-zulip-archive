Let C\mathsf{C}C be a strict 222-category; I'm going to write composition of both 111 and 222-cells with ∘\circ∘, write 111-cell identities as id⁡\operatorname{id}id, and write both left and right whiskering as a binary operator #\##. Suppose TTT is a strict 222-monad on C\mathsf{C}C (i.e. a Cat\mathsf{Cat}Cat-enriched monad) with unit η:Id⁡C→T\eta : \operatorname{Id}_{\mathsf{C}} \to Tη:IdC​→T and multiplication μ:T2→T\mu : T^2 \to Tμ:T2→T. For an object XXX of C\mathsf{C}C, define a TTT-prealgebra structure on XXX to be the data of a 111-cell θ:T(X)→X\theta : T(X) \to Xθ:T(X)→X and 222-cells ζ:id⁡X→θ∘ηX\zeta : \operatorname{id}_X \to \theta \circ \eta_Xζ:idX​→θ∘ηX​, ω:θ∘T(θ)→θ∘μX\omega : \theta \circ T(\theta) \to \theta \circ \mu_Xω:θ∘T(θ)→θ∘μX​. A prealgebra structure is a lax algebra if the data satisfies some coherence axioms. For another object YYY define a preadjunction from XXX to YYY to be the data of 111-cells g:X→Yg : X \to Yg:X→Y, f:Y→Xf : Y \to Xf:Y→X and 222-cells α:id⁡Y→g∘f\alpha : \operatorname{id}_Y \to g\circ fα:idY​→g∘f, β:f∘g→id⁡X\beta : f \circ g \to \operatorname{id}_Xβ:f∘g→idX​. Such a thing is an adjunction if α,β\alpha, \betaα,β satisfy the triangle identities. Given a prealgebra structure on XXX and a preadjunction from XXX to YYY I can define a prealgebra structure on YYY by

Kevin Arlin (Oct 23 2023 at 17:42):

I'm not strongly recommending it, but to flesh out one of your ideas: there is a simple explicit description for the codensity 2-monads: we have ⟨X,X⟩Y=C(Y,X)⋔X.\langle X,X\rangle Y=\mathsf C(Y,X)\pitchfork X.⟨X,X⟩Y=C(Y,X)⋔X. Now considering the lax morphism T→⟨X,X⟩,T\to \langle X,X\rangle,T→⟨X,X⟩, this rephrases your question as whether an adjunction induces a lax morphism ⟨X,X⟩→⟨Y,Y⟩.\langle X,X\rangle\to \langle Y,Y\rangle.⟨X,X⟩→⟨Y,Y⟩. Or, in other words again, you want to know whether this adjunction makes YYY into an XXX (i.e. an ⟨X,X⟩\langle X,X\rangle⟨X,X⟩-algebra. I don't immediately see that this is any better than checking the proposal you already have by hand, though.
