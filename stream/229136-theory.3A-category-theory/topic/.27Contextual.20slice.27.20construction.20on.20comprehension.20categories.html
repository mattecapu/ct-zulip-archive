Rafaël Bocquet (Mar 19 2025 at 13:34):

I am not aware of any reference for comprehension categories.

In the case of categories with families, the contextual slice can be defined as the contextual core of the slice CwF:

C//Γ=cxl(C/Γ) \mathcal{C} // \Gamma = \mathsf{cxl}(\mathcal{C} / \Gamma) C//Γ=cxl(C/Γ).

The contextual core cxl(C) \mathsf{cxl}(\mathcal{C}) cxl(C) of a CwF C \mathcal{C} C has the following universal property:
for every map F:D→CF : \mathcal{D} \to \mathcal{C} F:D→C with bijective actions on types and terms, the inclusion map cxl(C)→C \mathsf{cxl}(\mathcal{C}) \to \mathcal{C} cxl(C)→C factors uniquely through F F F. This can also be phrased using an orthogonal factorization system.
(AFAIK, the earliest reference for this is this note of Christian Sattler: https://www.cse.chalmers.se/~sattler/docs/normalization-by-evaluation.pdf)

Moreover, if C \mathcal{C} C itself is contextual, we have (C//Γ)≅(C[γ‾:1→Γ]) (\mathcal{C} // \Gamma) \cong (\mathcal{C}[\underline{\gamma} : 1 \to \Gamma]) (C//Γ)≅(C[γ​:1→Γ]), where (C[γ‾:1→Γ]) (\mathcal{C}[\underline{\gamma} : 1 \to \Gamma]) (C[γ​:1→Γ]) is the free extension of C \mathcal{C} C by a new morphism from the terminal object to Γ \Gamma Γ.

I would imagine that analogous universal properties can be proven for comprehension categories. The situation is a bit more complicated because comprehension categories form a 2-category while CwFs form a 1-category.
