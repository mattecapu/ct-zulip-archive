It seems that you have

F♯∈∫xSet(A(x),Set(B(x),C))F^{\sharp} \in \int_x Set(A(x),Set(B(x),C))F♯∈∫x​Set(A(x),Set(B(x),C))

(by characterization of natural transformations as an end) and also the series of iso:

∫xSet(A(x),Set(B(x),C))≅∫x∏A(x)Set(B(x),C)\int_x Set(A(x),Set(B(x),C)) \cong \int_x \prod_{A(x)}Set(B(x),C)∫x​Set(A(x),Set(B(x),C))≅∫x​∏A(x)​Set(B(x),C)

⋯≅∫xSet(∐A(x)B(x),C)\dots \cong \int_x Set(\coprod_{A(x)}B(x),C)⋯≅∫x​Set(∐A(x)​B(x),C)

⋯≅Set(∫xA(x)×B(x),C)\dots \cong Set(\int^xA(x)\times B(x),C)⋯≅Set(∫xA(x)×B(x),C)

Then, express injectiveness as a lifting problem in Set, or rather translate that characterization in homset language: i.e. a function X→YX\to YX→Y is injective if the cospan:

Set(2,X)→Set(2,Y)Set(\mathbf{2},X) \to Set(\mathbf{2},Y)Set(2,X)→Set(2,Y)

Set(1,Y)→Set(2,Y)Set(\mathbf{1},Y) \to Set(\mathbf{2},Y)Set(1,Y)→Set(2,Y)

admits Set(1,X)Set(\mathbf{1},X)Set(1,X) as a weak pullback (maybe there is a better way to say it).

Note: You can generalize the previous characterization with the composition map
Set(2,X)×Set(X,Y)→Set(2,Y)Set(\mathbf{2},X) \times Set(X,Y) \to Set(\mathbf{2},Y)Set(2,X)×Set(X,Y)→Set(2,Y)
to characterize the set of injections X→YX\to YX→Y. This characterization should probably look like a lifting problem for a certain diagram where you start by "choosing 1→Set(X,Y)\mathbf{1} \to Set(X,Y)1→Set(X,Y) such that ...".

After this, the sequence of isomorphisms I shown above should lead you to an answer (by successfully transferring -- or failing to transfer -- the lifting problem from one side to another)

Note: You may need to use some smallness property for 2\mathbf{2}2 and 1\mathbf{1}1 with respect to the colimit ∫x∐A(x)B(x)\int^x\coprod_{A(x)} B(x)∫x∐A(x)​B(x). Interestingly, this hints toward a condition involving both A(x)A(x)A(x) and B(x)B(x)B(x), hence a little more than just the domain of F♯F^{\sharp}F♯.

You can repeat the same argument for surjectiveness using the map 0→1\mathbf{0} \to \mathbf{1}0→1
