Topic: How is this "initial object" called?

Bruno Gavranović (Jun 09 2023 at 20:03):

I have a construction that resembles a lax/weak/local initial object and, because I'm not sure if a good reference for these things exists, I'm not sure how to disambiguate between them and determine what this is. Let me be precise.

I have a bicategory B\mathcal{B}B and an object I:BI : \mathcal{B}I:B such that a few different things hold:

  • For every object XXX, the hom-category B(I,X)\mathcal{B}(I, X)B(I,X) has an initial object iX:I→Xi_X: I \to XiX​:I→X (i.e. for every other other g:I→Xg : I \to Xg:I→X there is a unique 2-cell ix⇒gi_x \Rightarrow gix​⇒g);
  • For every f:X→Yf : X \to Yf:X→Y in B\mathcal{B}B there is a unique 2-cell iX;f⇒iYi_X ; f \Rightarrow i_YiX​;f⇒iY​

The second condition makes me think that this is a lax initial object, as the commuting condition isn't satisfied strictly.
On the other hand, I feel like there's a missing "...such that" condition to this structure that I haven't quite figured out yet.
Likewise, I'm not sure whether every "lax initial object" (whose definition I'm still uncertain of) will locally be a strict initial object, as I have here.

Bruno Gavranović (Jun 11 2023 at 18:11):

While talking to @Igor Bakovic I realised my problem specification is incorrect in a manner where it makes it clear what this construction is.
Note that the second bullet point has a map going into the initial object iyi_yiy​ which is a little bit suspicious. As it turns out, I made a mistake and my setting has a map going out of it. This then means that this bullet point is redudant.

Then, looking at the remaining bullet point, the condition of B(I,X)\mathcal{B}(I, X)B(I,X) having an initial object means that there exists a left adjoint to the terminal functor B(I,X)→1\mathcal{B}(I, X) \to \mathbf{1}B(I,X)→1.
This generalises the setting of an initial object in a category (where we have an isomorphism C(I,X)≅1\mathcal{C}(I, X) \cong 1C(I,X)≅1, with C\mathcal{C}C being any category) to one in a bicategory (where now we have an adjunction).

When we generalise isomorphisms to adjunctions this is often called a quasi-colimit . So my bicategory really has a quasi-initial object.

Some thoughts I had while realising this:

  • I don't understand why this is called a quasi limit and not a lax limit. The latter name seems more apt as we only have 2-cells going in one direction.
  • It looks like a quasi-initial object III in B\mathcal{B}B implies the existence of a strict colimit in every hom-category B(I,X)\mathcal{B}(I, X)B(I,X). I never heard about this fact before, but it sounds important to know.
  • More surprisingly, even, it seems like this holds the other way too. Having a strict colimit in every such hom-category B(I,X)\mathcal{B}(I, X)B(I,X) seems to imply that B\mathcal{B}B has a quasi-initial object, as postcomposition with f:X→Yf : X \to Yf:X→Y lands us in B(I,Y)\mathcal{B}(I, Y)B(I,Y) for which already have proof that it contains a map from the initial object to any other.

Mike Shulman (Jun 11 2023 at 18:15):

I believe the terminology "quasi" is due to Gray, who was writing before the standard meaning of "lax" as "up to a noninvertible transformation" had been adopted. I don't think these things have been studied further sufficiently for a consistent modern terminology to be adopted. But the terminology [[lax limit]] is not available because that already means something different.

Mike Shulman (Jun 11 2023 at 18:16):

(In particular, a "lax initial object" is the same as an ordinary initial object.)

Igor Bakovic (Jun 12 2023 at 17:02):

Jay gave an elementary description of the kind of conical limits of morphisms of bicategories which arise from local adjunctions in

C.B. Jay, Local adjunctions, Journal of Pure and Applied Algebra 53, (1988) 227-238.

In Example 2.7. he defines a conical limit of a morphism F ⁣:D→AF \colon \mathcal{D} \to \mathcal{A}F:D→A of bicategories by means of a local right adjoint to the diagonal (strict) homomorphism

Δ ⁣:A→Bicatop(D,A)\Delta \colon \mathcal{A} \to Bicat^{op}(\mathcal{D},\mathcal{A})Δ:A→Bicatop(D,A)

where Bicatop(D,A)Bicat^{op}(\mathcal{D},\mathcal{A})Bicatop(D,A) is the bicategory whose objects are morphisms of bicategories, 1-cells are oplax transformations and 2-cells are modifications. Then a limit for FFF is a lax cone α ⁣:Δ(X)→F\alpha \colon \Delta (X) \to Fα:Δ(X)→F such that, for each lax cone β ⁣:Δ(Y)→F\beta \colon \Delta (Y) \to Fβ:Δ(Y)→F there is a pair (f,σ)(f, \sigma)(f,σ) where f ⁣:Y→Xf \colon Y \to Xf:Y→X is a l-cell in A\mathcal{A}A and σ ⁣:α∘Δ(f)→β\sigma \colon \alpha \circ \Delta (f) \to \betaσ:α∘Δ(f)→β is a modification, which is universal among such pairs in the sense that for any other such pair (f′,σ′)(f', \sigma')(f′,σ′) there exists a unique 2-cell χ ⁣:f′→f\chi \colon f' \to fχ:f′→f such that σ′=σ(α∘Δ(χ))\sigma'=\sigma(\alpha \circ \Delta (\chi))σ′=σ(α∘Δ(χ)).

In the special case when D\mathcal{D}D is ∅\emptyset∅ a conical limit TTT of the empty diagram ∅→A\emptyset \to \mathcal{A}∅→A is an object with a property of a local terminal object - an object such that for each object AAA there is a terminal object in the hom-category A(A,T)\mathcal{A} (A, T)A(A,T).

Therefore, what you have is a dual situation - a local initial object.

I could say much more about the role of local initial and terminal objects in local and lax adjunctions but perhaps I should stop here for now.
