Topic: Faithful presheaf over presheaves

Paolo Perrone (Apr 21 2024 at 11:22):

Let FFF be a presheaf on a category C\mathrm{C}C (we can assume $$\cat{C}$$ small, if it helps).
Suppose that FFF, as a functor Cop→Set\mathrm{C}^\mathrm{op}\to\mathrm{Set}Cop→Set is faithful.

Consider now the representable presheaf on the category of presheaves [Cop,Set](−,F)[\mathrm{C}^\mathrm{op},\mathrm{Set}](-,F)[Cop,Set](−,F). Is this a faithful functor as well, on [Cop,Set][\mathrm{C}^\mathrm{op},\mathrm{Set}][Cop,Set]?
(Note that, by the Yoneda lemma, its restriction along the Yoneda embedding is FFF, which is faithful.)

Peva Blanchard (Apr 21 2024 at 13:34):

Let C=1C = 1C=1 the category with a single object. Then a presheaf on CCC is just a set, and is necessarily faithful. I choose FFF to be the singleton set {0}\{0\}{0}.

Now, let G={0,1}G = \{0,1\}G={0,1} be the two-element set considered as a (faithful) presheaf on CCC.

  • Let ϕ\phiϕ be the identity function on GGG considered as an element of [Cop,Set](G,G)[C^{op}, Set](G, G)[Cop,Set](G,G)
  • Let ψ\psiψ be the "swap" function on GGG, also considered as an element of [Cop,Set](G,G)[C^{op}, Set](G, G)[Cop,Set](G,G)
    Then ϕ\phiϕ and ψ\psiψ are different.

Pick any η:[Cop,Set](G,F)\eta : [C^{op}, Set](G, F)η:[Cop,Set](G,F). There is actually not much choice here: this natural transformation is equivalently described as the unique function {0,1}→{0}\{0, 1\} \rightarrow \{0\}{0,1}→{0}. We have in particular:

η∘ϕ=η∘ψ \eta \circ \phi = \eta \circ \psiη∘ϕ=η∘ψ

I think this shows that the representable [Cop,Set](_,F)[C^{op}, Set](\_, F)[Cop,Set](_,F) is not faithful.

Mmh, it does not help either.

Take for instance CCC the category freely generated by the graph with one object ⋆\star⋆ and one non-trivial morphism s:⋆→⋆s : \star \rightarrow \stars:⋆→⋆.

A presheaf FFF on CCC is just a set XXX with a function f:X→Xf : X \rightarrow Xf:X→X. The presheaf is faithful iff f≠idf \ne idf=id. Let's choose F=NF = \mathbb{N}F=N with f(n)=n+1f(n) = n + 1f(n)=n+1.

Now, let GGG be another presheaf on CCC, given by a set YYY and a function g:Y→Yg : Y \rightarrow Yg:Y→Y.

Assume there exists a natural transformation η:C^(G,F)\eta : \hat{C}(G, F)η:C^(G,F). This means that we have a function h:Y→Nh : Y \rightarrow \mathbb{N}h:Y→N s.t. h(g(y))=h(y)+1h(g(y)) = h(y) + 1h(g(y))=h(y)+1 for all yyy. This implies, in particular, that YYY is the empty set or infinite.

Stated otherwise, if YYY is a non-empty finite set, then C^(G,F)=∅\hat{C}(G, F) = \emptysetC^(G,F)=∅. In that case, any ψ:C^(G,G)\psi : \hat{C}(G, G)ψ:C^(G,G) satisfies

∀η:C^(G,F),η∘ψ=η∘id \forall \eta: \hat{C}(G, F), \eta \circ \psi = \eta \circ id∀η:C^(G,F),η∘ψ=η∘id

Thus, it suffices to find a ψ≠id\psi \ne idψ=id to prove that the representable C^(_,F)\hat{C}(\_, F)C^(_,F) is not faithful.

Consider the presheaf GGG with Y={0,1}Y = \{0, 1\}Y={0,1} and g=swapg = \text{swap}g=swap. Let ψ=swap\psi = \text{swap}ψ=swap. This choice defines a natural transformation in C^(G,G)\hat{C}(G, G)C^(G,G) since ψ\psiψ commutes with ggg. And ψ≠id\psi \ne idψ=id.

Therefore C^(_,F)\hat{C}(\_, F)C^(_,F) is not faithful.

Let CCC be the full sub-category of SetSetSet spanned by the sets S={0}S = \{0\}S={0} and U={0,1}U = \{0,1\}U={0,1}, depicted on the left in the picture above. If I am not mistaken, SSS is a separator in CCC.

Let F,GF, GF,G be presheaves on CCC defined as follows.

FU=GU=UFS=GS=SFv0=Fv1=unique map {0,1}→{0}F!=constant 0Gv0=Gv1=unique map {0,1}→{0}G!=constant 1\begin{align*} FU &= GU = U \\ FS &= GS = S \\ Fv_0 &= Fv_1 = \text{unique map } \{0,1\} \rightarrow \{0\} \\ F! &= \text{constant } 0\\ Gv_0 &= Gv_1 = \text{unique map } \{0,1\} \rightarrow \{0\} \\ G! &= \text{constant } 1\\ \end{align*}FUFSFv0​F!Gv0​G!​=GU=U=GS=S=Fv1​=unique map {0,1}→{0}=constant 0=Gv1​=unique map {0,1}→{0}=constant 1​

There are two distinct natural transformations F⇒GF \Rightarrow GF⇒G with the same component at SSS. Their component at UUU is swap\text{swap}swap or the constant 111 respectively.

This shows that YSYSYS is not a separator in C^\hat{C}C^.
