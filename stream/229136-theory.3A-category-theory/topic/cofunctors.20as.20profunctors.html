For comparison I decided to try to describe representable profunctors in a similar way:

A representable profunctor is a profunctor P:A⇸BP : A ⇸ BP:A⇸B equipped with the following data:

  • for each a∈Aa \in Aa∈A, an object bab_aba​ of BBB and an element pap_apa​ of P(ba,a)P(b_a,a)P(ba​,a)
  • such that each element ppp of any P(b,a)P(b,a)P(b,a) appears as P(f,id) paP(f,id)\,p_{a}P(f,id)pa​ for some unique f:b→baf : b \to b_af:b→ba​.

From this data we can reconstruct a functor F:A→BF : A \to BF:A→B as follows:

  • F0(a)=baF_0(a) = b_aF0​(a)=ba​
  • F1(f)F_1(f)F1​(f) for f:a→a′f : a \to a'f:a→a′ is the unique ggg such that P(id,f) pa=P(g,id) pa′P(id,f)\,p_a = P(g,id)\,p_{a'}P(id,f)pa​=P(g,id)pa′​.

  • P(b,a)=B(b,F0(a))P(b,a) = B(b,F_0(a))P(b,a)=B(b,F0​(a))
