Mike Shulman (Apr 27 2020 at 23:08):

Well, the ∗\ast∗-autonomous structure of double-glued categories is constructed from the analogous structure of the input categories plus limits. So I would expect that if you have a functor that preserves all that structure, including the relevant limits, then it should induce a structure-preserving functor on the double-glued categories.

Andrea Schalk (May 06 2020 at 13:25):

Is this question still open? For the first case you give, to define the functor you need to make a choice, I think. Assume you have (R,U,X)(R,U,X)(R,U,X) in the tight orthogonality category for C\mathcal{C}C. While it is clear that (FR,F[U],F[X])(FR,F[U],F[X])(FR,F[U],F[X]) will be an object in the slack orthogonality category for D\mathcal{D}D, in order to ensure you obtain something in the tight subcategory you either have to ask for extra properties for FFF, or you have to apply a closure operation, but then you have to make a choice between using (FR,(F[U])00,(F[U])0)(FR,(F[U])^{00},(F[U])^0)(FR,(F[U])00,(F[U])0) and (FR,(F[X])0,(F[X])00)(FR,(F[X])^0,(F[X])^{00})(FR,(F[X])0,(F[X])00).
