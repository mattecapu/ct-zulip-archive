Topic: fiberwise homeomorphisms

Jens Hemelaer (Jan 14 2021 at 16:38):

If f:E→E′f : E \to E'f:E→E′ is a continuous map of étale spaces over some base topological space BBB, then it is an homeomorphism if and only if it is a fiberwise homeomorphism. (This is a translation of "a morphism of sheaves is an isomorphism if it is an isomorphism in each stalk".)

Are there any generalizations to this where the spaces EEE and E′E'E′ are no longer étale? For example, is it enough to assume that the structure maps π:E→B\pi : E \to Bπ:E→B and π:E′→B\pi : E' \to Bπ:E′→B are both open?

In any case, if f:E→E′f : E \to E'f:E→E′ is a fiberwise homeomorphism, then it is a bijection. So the difficulty is in showing that fff is open.

Jens Hemelaer (Jan 15 2021 at 10:54):

If EEE and E′E'E′ are both topological manifolds of the same dimension (not necessarily connected), then any continuous bijection f:E→E′f : E \to E'f:E→E′ is automatically a homeomorphism (by invariance of domains). So in this case, it is enough to check bijectivity, or equivalently, bijectivity in each fiber.

I was hoping for results that work for general topological spaces, but this already excludes some counterexamples.

Jens Hemelaer (Jan 20 2021 at 09:21):

@Ming Ng If EEE is a space with a GGG-action, together with a map p:E→Bp : E \to Bp:E→B such that p(g⋅e)=p(e)p(g \cdot e) = p(e)p(g⋅e)=p(e), then EEE is a GGG-torsor if and only if the map
ξ:G×E→E×BE, (g,x)↦(gx,x)\xi : G \times E \to E \times_B E,~ (g,x) \mapsto (gx,x)ξ:G×E→E×B​E, (g,x)↦(gx,x)
is a homeomorphism. It follows in particular that the action is free, and that the action is transitive on the fibers.
Further, there are natural projection maps π2:G×E→E\pi_2 : G \times E \to Eπ2​:G×E→E and π2′:E×BE→E\pi_2': E \times_B E \to Eπ2′​:E×B​E→E, and for these projection maps we have that π2′∘ξ=π2\pi_2' \circ \xi = \pi_2π2′​∘ξ=π2​. So ξ\xiξ is a map "over" EEE.

The question is now, is it enough that ξ\xiξ is a fiberwise homeomorphism to deduce that it must be a homeomorphism? This would simplify a lot, because on the fibers over some point x∈Ex \in Ex∈E, the map ξ\xiξ is given by the map
ξ:G→O(x), g↦g⋅x\xi : G \to \mathcal{O}(x),~ g \mapsto g \cdot xξ:G→O(x), g↦g⋅x,
where O(x)\mathcal{O}(x)O(x) denotes the orbit of xxx.

So in order for ξ\xiξ to be a fiberwise homeomorphism, it is enough that in EEE, each orbit is (in a natural way) homeomorphic to GGG!

Unfortunately, if a map is a fiberwise homeomorphism that doesn't necessarily mean it is a homeomorphism, see for example the map fff from above.

On the other hand, if EEE and E′E'E′ are both GGG-torsors over BBB, and g:E→E′g : E \to E'g:E→E′ is any GGG-equivariant map over BBB, then I think ggg is necessarily a homeomorphism (non-isomorphic torsors don't have maps between them). In the example above, f:R×[0,+∞)→Mf : \mathbb{R} \times [0,+\infty) \to Mf:R×[0,+∞)→M is an R\mathbb{R}R-equivariant map over [0,+∞)[0,+\infty)[0,+∞), which is not a homeomorphism. The domain is a (trivial) torsor over [0,+∞)[0,+\infty)[0,+∞), so this means that the codomain can't be a torsor (despite the orbits being all naturally homeomorphic to R\mathbb{R}R).
