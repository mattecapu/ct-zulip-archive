Topic: power objects up to internal equivalence

Tobias Fritz (Aug 08 2021 at 06:55):

Given a finitely complete category C\mathcal{C}C, a power object for A∈CA \in \mathcal{C}A∈C is an object PAPAPA which represents the presheaf Sub(A×−)\mathsf{Sub}(A \times -)Sub(A×−). Unfolding this definition via the Yoneda lemma gives the equivalent definition of a power object: it comes equipped with a distinguished monomorphism ∈ ⊆A×PA\in \, \subseteq A \times PA∈⊆A×PA satisfying the usual property.

I've just come across the following slightly weaker notion of power object, and I wonder if this has been considered before. I'm also looking for examples that would not be power objects in the usual sense.

Note first that every Sub(X)\mathsf{Sub}(X)Sub(X) is actually a partially ordered set; in fact, if we care about this poset only up to equivalence of preorders, then there is no need to define a subobject of XXX as an isomorphism class of monomorphisms into XXX, and we may just as well do the simpler thing of defining Sub(X)\mathsf{Sub}(X)Sub(X) as the collection of all monomorphisms into XXX, which makes it into a preordered (large) set with preorder given as usual with respect to factorization. My point is only to indicate that I care about Sub(X)\mathsf{Sub}(X)Sub(X) only up to equivalence of preordered sets.

Now let's say that a poweroid object for AAA is an internally preordered object (PA,≤)(PA,\le)(PA,≤), by which I mean that ≤↪PA×PA\le \hookrightarrow PA \times PA≤↪PA×PA is reflexive and transitive, which represents the subobject functor in the sense that there is a (pseudo)natural equivalence
C(−,PA)≅Sub(A×−),\qquad \qquad \mathcal{C}(-,PA) \cong \mathsf{Sub}(A \times -),C(−,PA)≅Sub(A×−),
where the left-hand side is also a preordered set in the obvious way.

I can show that a power object is the same thing as a poweroid object for which ≤\le≤ is internally antisymmetric (though I should double-check the proof). Now here are my questions:

  • Is there an object AAA in a finitely complete category which has a poweroid object but no power object? The obvious guess is that such a thing must exist, but finding an example seems to be quite tricky. I think that it will have to live in a finitely complete category that is not exact. Any ideas?

  • What are poweroid objects really called, and where can I read about them?

Mike Shulman (Aug 08 2021 at 16:24):

I don't have a complete answer, but one thing to note is that the notion of poweroid object uniquely determines the relation ≤\le≤, while if C\mathcal{C}C is a Heyting category this relation can be constructed. Thus, in a Heyting category, a poweroid object is just a plain object PAPAPA together with a natural transformation C(−,PA)→Sub(A×−)\mathcal{C}(-,PA) \to \mathsf{Sub}(A\times -)C(−,PA)→Sub(A×−) that is pointwise essentially surjective; we can then use this to define ≤\le≤ making PAPAPA an internal preorder. Thus, a poweroid object in a Heyting category is equivalently a "generic subobject of AAA", where "generic" is used in the sense of "classifying up to existence but not uniqueness" — every subobject of A×BA\times BA×B is classified, up to isomorphism, by some map B→PAB\to PAB→PA, but not necessarily a unique one.

Mike Shulman (Aug 08 2021 at 16:27):

Relatedly, a tripos is a first-order hyperdoctrine (i.e. a fibration of Heyting algebras with simple indexed products and coproducts) that has "generic predicates" in a sense that generalizes poweroid objects. Put differently, a Heyting category has all poweroid objects if and only if its subobject fibration is a tripos.

Mike Shulman (Aug 08 2021 at 16:31):

This emphasizes the connection to exactness: the category of PERs in a tripos is a topos, so the ex/reg completion of a category with poweroid objects will have power objects.

Mike Shulman (Aug 08 2021 at 16:54):

It also suggests one possible source of categories with poweroid objects: start with a tripos and do half of the tripos-to-topos construction. I haven't checked that this works, but my guess would be that if you start with a tripos and split only the coreflexive morphisms in its bicategory of relations, you would get a category with poweroid objects. For instance, you could investigate categories of assemblies.

Mike Shulman (Aug 09 2021 at 17:09):

@Tobias Fritz Note that your category of pairs is the category of separated objects for a Lawvere-Tierney topology on the topos Set→\rm Set^{\to}Set→. In particular, it is a quasitopos, so it has a strong-subobject classifier (namely (2,2)(2,2)(2,2)). On the other hand there is also the actual subobject classifier of the ambient topos, "(2,3)(2,3)(2,3)", which is not separated. I think your observation is that there's a strong epimorphism (5,3)→(2,3)(5,3) \to (2,3)(5,3)→(2,3) (actually wouldn't (3,3)(3,3)(3,3) work as well?) and that your category satisfies an axiom of choice with respect to this strong epi. Does that seem right? If so, it seems unlikely to generalize to other toposes, since having this axiom of choice seems likely to depend on the ordinary axiom of choice, or at least the law of excluded middle, in Set.
