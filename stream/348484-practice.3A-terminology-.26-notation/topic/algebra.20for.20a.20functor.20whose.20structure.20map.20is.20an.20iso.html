Topic: algebra for a functor whose structure map is an iso

Nathaniel Virgo (Aug 12 2024 at 05:21):

Is there a name for an algebra for a functor whose structure map is an isomorphism, or equivalently a coalgebra whose structure map is an isomorphism? Initial algebras and final coalgebras have this property, but there can be others as well.

The obvious name would be [[bialgebra]], but that's taken for something else.

Nathanael Arkor (Aug 12 2024 at 06:56):

These tend to be called [[fixed points]].
