  • note that q.uiver.app is presently not able to export to SVG
    https://github.com/varkor/quiver/issues/8

  • arrowgram is a React component that renders SVG diagrams of arrows, nodes, and labels from a declarative JSON specification. It is designed for creating mathematical diagrams, such as commutative diagrams used in category theory, directly within a web application. The component is self-contained, performing all necessary geometric calculations to translate the abstract specification into a visually clear SVG representation. It is intended for use in markdown rendering pipelines or interactive development environments like Sandpack. arrowgram can export to SVG.
