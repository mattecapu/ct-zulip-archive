There is lots of affine categories which are funnier that an affine space. The idea is that now, the objects of the affine category are still the points of our space but there is way more possibilities for vectors between these points.

Jean-Baptiste Vienney (Jul 24 2022 at 16:42):

Now we can have different morphisms a→ba \rightarrow ba→b not only one vector. These morphisms are thus more paths than vectors. In an affine space, there is only one such path which is given by ab→\overset{\rightarrow}{ab}ab→.

Jean-Baptiste Vienney (Jul 24 2022 at 16:46):

However, mere topological spaces are not affine categories. Because there is an idea of uniformity in an affine category: the paths starting from any point aaa are the same than the paths starting from any point bbb.

Jean-Baptiste Vienney (Jul 24 2022 at 16:48):

Example:
Rn\mathbb{R}^{n}Rn is an affine category for every n≥1n \ge 1n≥1. A morphism a→ba \rightarrow ba→b is given by an equivalence class f‾\overline{f}f​ where fff is a path from aaa to bbb and under the equivalence relation ∼\sim∼ on the set of all paths from any point to any point (considered as continuous functions [0,1]→Rn[0,1] \rightarrow \mathbb{R}^{n}[0,1]→Rn) given by f∼gf \sim gf∼g iff f−gf-gf−g is a constant. It means that a morphism a→ba \rightarrow ba→b is a path from aaa to bbb but when you translate to the same path a′→b′a' \rightarrow b'a′→b′ it's always considered as the same morphism.

Jean-Baptiste Vienney (Jul 24 2022 at 16:58):

We can see with this example that in an affine category, MorCMor_{\mathcal{C}}MorC​ is not necessarily a commutative monoid such as in an affine space. When you compose two vectors in an affine space, the order doesn't matter because it is a very simple case.
