Topic: NixCon 2020

Nick Bathum (Sep 23 2020 at 01:21):

Event Details:

  • Friday October 16th through the 18th.
  • Talks run roughly 11:00 to 18:00 UTC
  • Join online https://2020.nixcon.org/, powered by Free software and NixOS.

If you write Haskell or Rust code, or often link with a lot of native dependencies, you should at least be aware of Nix.

Nix is a package manager for (some) *nix systems that applies a content-addressed viewpoint to the concept of a software package. So packages are compositions of other packages, which gives nice properties for deploying software. See some descriptions for Nix and NixOS here: https://nixos.org/

Although there are no Haskell talks on the schedule (as of this posting anyway), there is a "hallway" option available by way of creating a room in Jitsi and I'd be surprised if you couldn't find interested Nix+Haskell people to interact with. If you have any questions you can contact me on Zulip or email orgateam@nixcon.org. :heart:
