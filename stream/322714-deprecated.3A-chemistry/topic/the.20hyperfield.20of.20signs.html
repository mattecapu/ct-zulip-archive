John Baez (May 27 2022 at 03:18):

It might be fun to interpret motifs as giving matrices in the hyperfield of signs. This is a number system with 3 elements,

S={+,0,−} \mathbb{S} = \{+, 0, - \}S={+,0,−}

where you add, multiply, subtract and divide by nonzero elements in the "obvious way", taking into account the fact that addition can be multivalued, namely +\mathbf{+}+ plus −\mathbf{-}− takes all 3 values, +,0,−\mathbf{+}, 0, \mathbf{-}+,0,−. This is the only case where we need the multivaluedness.
