Oscar Cunningham (May 19 2020 at 11:27):

I also managed to get LaTeX\LaTeXLATE​X working nicely in the comments, including a live preview of what people have typed. The trick was to install the WP-Markdown plugin, and then use the Plugin Editor to add the line editor.hooks.chain("onPreviewRefresh", MathJax.typeset); just before editor.run(); in the file js/markdown.js. You then have to run this file through a JavaScript minifier and use the result to replace the file js/markdown.min.js. (I believe that for this to work you might have to have your MathJax version set to 3 in the settings of Simple Mathjax, but this is the most recent version anyway.)
