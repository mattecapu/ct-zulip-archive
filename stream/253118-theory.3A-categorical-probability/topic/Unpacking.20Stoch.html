Stream: theory: categorical probability

Topic: Unpacking Stoch

Nathaniel Virgo (Feb 25 2023 at 11:23):

I'm trying to unpack the definition of Stoch (the Kleisli category of the Giry monad) as a Markov category. My goal is to be able to take any string diagram and be able to express it explicitly in terms of measure-theoretic integrals, in order to prove properties of Stoch. I don't know much measure theory and am trying to learn what I need as I go along.

So far the best resource I've found for unpacking Stoch is Panangaden's 'The Category of Markov Kernels' (1999), which is really helpful but stops short of completely answering my question. The tricky part is in the details of how to reason about the interaction between the monoidal product and composition, as I explain below.

Here is what I know so far:

Objects are measurable spaces, which I'll write as (X,ΣX)(X,\Sigma_X)(X,ΣX​). I'll write σX,τX\sigma_X, \tau_XσX​,τX​ etc. for elements of ΣX\Sigma_XΣX​.

Then a morphism f ⁣:X→Yf\colon X\to Yf:X→Y is a map f ⁣:ΣY×X→[0,1]f\colon \Sigma_Y\times X\to [0,1]f:ΣY​×X→[0,1], where we write f(σY∣x)f(\sigma_Y\mid x)f(σY​∣x) for f(σY,x)f(\sigma_Y,x)f(σY​,x). This must have the properties that

  • f(−∣x) ⁣:ΣY→[0,1]f({-}\mid x)\colon \Sigma_Y\to [0,1]f(−∣x):ΣY​→[0,1] is a probability measure for every x∈Xx\in Xx∈X

  • f(B∣−) ⁣:X→[0,1]f(B\mid{-})\colon X\to [0,1]f(B∣−):X→[0,1] is a measurable function for every B∈ΣYB\in\Sigma_YB∈ΣY​.

Composition X→fY→gZX\xrightarrow{f}Y\xrightarrow{g}ZXf​Yg​Z is given by

(f;g)(σZ∣x)=∫y∈Yg(σZ∣y) d f(y∣x). (f;g)(\sigma_Z\mid x) = \int_{y\in Y} g(\sigma_Z\mid y) \,d\, f(y\mid x).(f;g)(σZ​∣x)=∫y∈Y​g(σZ​∣y)df(y∣x).

There is some work needed to see that this function has the needed properties and that composition is associative. Panangaden gives an explicit proof of associativity. Identities are Dirac deltas.

The monoidal product is given on objects by (X×Y,ΣX⊗ΣY)(X\times Y, \Sigma_X\otimes \Sigma_Y)(X×Y,ΣX​⊗ΣY​), where ΣX⊗ΣY\Sigma_X\otimes \Sigma_YΣX​⊗ΣY​ is the product σ\sigmaσ-algebra. For morphisms f ⁣:A→Xf\colon A\to Xf:A→X and g ⁣:B→Yg\colon B\to Yg:B→Y we define

(f⊗g)(σX×σY∣a,b)=f(σX∣a)g(σY∣b). (f\otimes g)(\sigma_{X}\times \sigma_{Y}\mid a,b) = f(\sigma_{X}\mid a)g(\sigma_{Y}\mid b).(f⊗g)(σX​×σY​∣a,b)=f(σX​∣a)g(σY​∣b).

This defines the measure only on elements of ΣX⊗ΣY\Sigma_X\otimes \Sigma_YΣX​⊗ΣY​ that are of the form σX×σY\sigma_{X}\times \sigma_{Y}σX​×σY​, but this uniquely extends to a measure on ΣX⊗ΣY\Sigma_X\otimes \Sigma_YΣX​⊗ΣY​ by Carathéodory's extension theorem.

So far so good. The tricky part is in figuring out how composition and this monoidal product relate to each other.

For example, suppose I have morphisms f ⁣:A→Xf\colon A\to Xf:A→X, g ⁣:B→Yg\colon B\to Yg:B→Y and h ⁣:X⊗Y→Zh\colon X\otimes Y\to Zh:X⊗Y→Z, and I want to calculate (f⊗g)⨾h(f\otimes g) \mathbin{⨾} h(f⊗g)⨾h. The morphism f⊗gf\otimes gf⊗g is as defined above, but in order to compose it with hhh I need to be able to integrate against it - I can't see a way to write (f⊗g)⨾h(f\otimes g) \mathbin{⨾} h(f⊗g)⨾h down as a single equation because of the step where we extend the partially defined measure to a full one.

Ultimately I want to be able to prove that certain morphisms are equal under certain conditions, and this sort of thing is causing a roadblock to that. So I guess my questions are

1) can the morphism (f⊗g)⨾h(f\otimes g) \mathbin{⨾} h(f⊗g)⨾h be written down explicitly in a nice way?

2) is there somewhere where this explicit unpacking of Stoch is done and written down already?

If these questions get answers I might have more questions as well, but let's see how this goes.

You could write it down as an integral. For all measurable

p:Z→[0,∞]p: Z \to [0, \infty]p:Z→[0,∞]

we have

∫((f⊗g);h)(a,b,dz)p(z)=∫f(a,dx)∫g(b,dy)∫h(x,y,dz) p(z).\int ((f \otimes g); h)(a, b, \text{d}z) p(z) = \int f(a, \text{d}x) \int g(b, \text{d}y) \int h(x, y, \text{d}z) \, p(z).∫((f⊗g);h)(a,b,dz)p(z)=∫f(a,dx)∫g(b,dy)∫h(x,y,dz)p(z).

Hence,

((f⊗g);h)(a,b,dz)=f(a,dx)g(b,dy)h(x,y,dz)((f \otimes g); h)(a, b, \text{d}z) = f(a, \text{d}x) g(b, \text{d}y) h(x, y, \text{d}z)((f⊗g);h)(a,b,dz)=f(a,dx)g(b,dy)h(x,y,dz)

using the notation in https://link.springer.com/book/10.1007/978-0-387-87859-1

Sorry if this is a bit brief, but I think that the explicit formula should simply be

((f⊗g);h)(S∣a,b)=∫x∈X∫y∈Yh(S∣x,y) df(x∣a) dg(y∣b).((f \otimes g) ; h)(S|a,b) = \int_{x \in X} \int_{y \in Y} h(S|x,y) \, df(x|a) \, dg(y|b).((f⊗g);h)(S∣a,b)=∫x∈X​∫y∈Y​h(S∣x,y)df(x∣a)dg(y∣b).

This is basically the same equation as calculating the expectation value of a measurable function with respect to a product measure. BTW I personally prefer writing this as
