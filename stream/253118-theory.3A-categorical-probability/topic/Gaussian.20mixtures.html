Tobias Fritz (May 20 2022 at 16:58):

So here's a conjectural construction which would let us construct a Markov category of Gaussian mixture models, but at the same time would apply way more generally. Before I state it, let me motivate it a bit and provide some intuition. For a Markov category of Gaussian mixture model, we presumably want the morphisms to be probability distributions over morphisms in Gauss\mathsf{Gauss}Gauss. So is there a general construction that lets us start with a Markov category C\mathsf{C}C and construct a new Markov category whose morphisms are probability distributions over morphisms of C\mathsf{C}C?

Here's how I think this works. Let C\mathsf{C}C be any [[commutative monad]] on Set\mathsf{Set}Set (or whatever category C\mathsf{C}C is enriched in), and suppose that TTT is affine, which means T1≅1T1 \cong 1T1≅1. Now define a new category TCT\mathsf{C}TC with the same objects as C\mathsf{C}C, but where (TC)(X,Y):=T(C(X,Y))(T\mathsf{C})(X,Y) := T(\mathsf{C}(X,Y))(TC)(X,Y):=T(C(X,Y)). For composition, we should be able to use
T(C(X,Y))×T(C(Y,Z))⟶T(C(X,Y)×C(Y,Z))→T(C(X,Z)),T(\mathsf{C}(X,Y)) \times T(\mathsf{C}(Y,Z)) \longrightarrow T(\mathsf{C}(X,Y) \times \mathsf{C}(Y,Z)) \to T(\mathsf{C}(X,Z)),T(C(X,Y))×T(C(Y,Z))⟶T(C(X,Y)×C(Y,Z))→T(C(X,Z)),
where the first morphism is the laxator of TTT and the second one induced by composition in C\mathsf{C}C. The monoidal structure should be defined similarly, and TCT\mathsf{C}TC should also inerhit the Markov structure from C\mathsf{C}C.

Conjecture: This construction makes sense and makes TCT\mathsf{C}TC into a Markov category.

Would you want to think about it? If we now take TTT to be the usual distribution monad and take C\mathsf{C}C to be Gauss\mathsf{Gauss}Gauss, then we should get a Markov category for Gaussian mixture models.
