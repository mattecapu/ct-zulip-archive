Topic: Tukey's Lemma

সায়ন্তন রায় (Dec 12 2020 at 14:53):

A family of sets F\mathcal {F}F is of finite character provided it has the following property: a set AAA belongs to F\mathcal{F}F iff every finite subset of AAA belongs to A\mathcal{A}A.

Tukey's lemma is the following statement,

Tukey's Lemma. Let ZZZ be a set and let F⊆P(Z)\mathcal {F}\subseteq {\mathcal {P}}(Z)F⊆P(Z). If F\mathcal {F}F is of finite character and X∈FX\in \mathcal {F}X∈F, then there is a maximal Y∈FY\in \mathcal {F}Y∈F (according to the inclusion relation) such that X⊆YX\subseteq YX⊆Y.

It is known that Tukey's Lemma is equivalent to Axiom of Choice.

Let us now call a family of sets F\mathcal {F}F to be of pseudo-finite character provided it has the following property: for any set AAA, if A∈FA\in \mathcal{F}A∈F then so does every finite subset of AAA. Consider the following statement,

Let ZZZ be a set and let F⊆P(Z)\mathcal {F}\subseteq {\mathcal {P}}(Z)F⊆P(Z). If F\mathcal {F}F is of pseudo-finite character and X∈FX\in \mathcal {F}X∈F, then there is a maximal Y∈FY\in \mathcal {F}Y∈F (according to the inclusion relation) such that X⊆YX\subseteq YX⊆Y.

Is the above statement equivalent to Tukey's Lemma? Clearly, this statement implies Tukey's Lemma. Does it strictly imply Tukey's Lemma?

Morgan Rogers (he/him) (Dec 12 2020 at 14:59):

Isn't the pseudo-finite version false? Let Z be the natural numbers, and let F\mathcal{F}F be the collection of finite subsets of ZZZ, which clearly is of pseudo-finite character. Then there is no maximal member of F\mathcal{F}F, whence there is in particular no maximal member containing ∅\emptyset∅.

সায়ন্তন রায় (Dec 19 2020 at 14:12):

Your suggestion worked. Thanks @[Mod] Morgan Rogers. The definition on which I finally settled is the following (I hope I have got it right this time):

Let ZZZ be a set. Call a family of sets F\mathcal {F}F to be of pseudo-finite character provided F=P(Z)\mathcal{F}=\mathcal{P}(Z)F=P(Z) or it has the following properties:

  • If W∈FW\in \mathcal{F}W∈F then P(W)⊆F\mathcal{P}(W)\subseteq \mathcal{F}P(W)⊆F.
  • There exists α∈Z\alpha\in Zα∈Z such that {α}∉F\{\alpha\}\notin \mathcal{F}{α}∈/F.

Then the pseudo-finite version of Tukey's Lemma is the following:

Let ZZZ be a set and let F⊆P(Z)\mathcal {F}\subseteq {\mathcal {P}}(Z)F⊆P(Z). If F\mathcal {F}F is of pseudo-finite character and X∈FX\in \mathcal {F}X∈F, then there is a maximal Y∈FY\in \mathcal {F}Y∈F (wrt the inclusion relation) such that X⊆YX\subseteq YX⊆Y.

Reid Barton (Dec 19 2020 at 15:00):

I don't think this is getting any closer to being correct. You could still take F\mathcal{F}F to be the collection of all finite subsets of ZZZ (an infinite set) which don't contain some fixed element α\alphaα.

Reid Barton (Dec 19 2020 at 15:02):

Think about how the proof has to work. If you start from an X∈FX \in \mathcal{F}X∈F which is not maximal, it means you can add an element to XXX to get X′X'X′ which is still in F\mathcal{F}F. If X′X'X′ is still in F\mathcal{F}F, you can add another element, and so on. Suppose this goes on forever. Then you need some way to deduce that the union of XXX, X′X'X′, ... is still in F\mathcal{F}F.

Reid Barton (Dec 19 2020 at 15:04):

The "finite character" hypothesis provides this because a finite subset of this union is a finite subset of some finite stage. But your "pseudo-finite character" hypothesis only gives you statements of the form: if a set belongs to F\mathcal{F}F, then a smaller set belongs to F\mathcal{F}F, which can never help.
