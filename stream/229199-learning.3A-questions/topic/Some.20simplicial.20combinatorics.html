Here's my proof. A map f:n→mf:n\to mf:n→m partitions nnn into a family of fibers; let S=n/∼S = n/\simS=n/∼ where a∼ba\sim ba∼b iff f(a)=f(b)f(a)=f(b)f(a)=f(b) . Given σ\sigmaσ a permutation of nnn, I construct a directed graph GσG_\sigmaGσ​. Formally GσG_\sigmaGσ​ is a function S×S→NS\times S\to \mathbb{N}S×S→N.
The edge from A to B is weighted by number of elements of A that σ\sigmaσ sends into B, the cardinality of A∩σ−1(B)A\cap \sigma^-1(B)A∩σ−1(B). The cardinality of A is equal to the sum of all edge weights for arrows leaving A, and also equal to the sum of edge weights for arrows entering A, so if arrows entering A are weighted negatively then the net flow into a node is zero. (I. e. if GσG_\sigmaGσ​ is viewed as a matrix the sum of any row is equal to the sum of its transpose column. Gσ1=Gσ⊤1G_\sigma \mathbf{1}=G_\sigma^\top \mathbf{1}Gσ​1=Gσ⊤​1.

The nodes of the graph are totally ordered in the obvious way.

Note that if there is an edge from A to B with positive weight and B<A, f\circ \sigma is not monotonic. So we assume that if B<A, the edge from A to B has zero weight. The matrix GσG_\sigmaGσ​ is upper triangular.
It is now easy to see by induction on rows that Gσ1=Gσ⊤1G_\sigma 1=G_\sigma^\top 1Gσ​1=Gσ⊤​1 forces GσG_\sigmaGσ​ to be lower triangular as well, i. e. concentrated along the diagonal. So fσ=ff\sigma =ffσ=f.
