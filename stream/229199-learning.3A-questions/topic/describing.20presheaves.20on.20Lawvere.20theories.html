Topic: describing presheaves on Lawvere theories

John Baez (Apr 10 2021 at 21:12):

@Christian Williams and I are trying to figure out how describe all the presheaves on a Lawvere theory T in an intuitive, tractable way. We came up with an idea and I'm wondering if someone can help polish it up.

I'll illustrate this idea with the Lawvere theory for monoids, though you can replace the word "monoid" in what follows by any other concept described by a Lawvere theory. (I like the method of "pretending to be doing an example when you're actually doing something general" - it's psychologically helpful, since it reduces the burden of abstraction.)

So, let TTT be the Lawvere theory of monoids. We're trying to describe presheaves on TTT.

We think of these as functors F:Top→SetF: T^{\text{op}} \to \mathsf{Set}F:Top→Set. Using a famous result of Lawvere, TopT^{\text{op}}Top is a skeleton of the category of free finitely generated monoids. So, there's a full and faithful functor

i:Top→Mon i: T^{\text{op}} \to \mathsf{Mon} i:Top→Mon

So, we can get a presheaf FFF on TTT from any functor

G:Mon→Set G : \mathsf{Mon} \to \mathsf{Set} G:Mon→Set

simply by composing with iii:

F=G∘i F = G \circ iF=G∘i

I claim that any presheaf on TTT arises this way.

Next, how do we get lots of functors

G:Mon→Set? G : \mathsf{Mon} \to \mathsf{Set} ?G:Mon→Set?

Here's an example:

G(X)={(x,y)∈X2:x2=y3}(x,y)∼(y,1) G(X) = \displaystyle{ \frac{\{(x,y) \in X^2 : x^2 = y^3\}}{(x,y) \sim (y,1)} } G(X)=(x,y)∼(y,1){(x,y)∈X2:x2=y3}​

Here I'm building a set from a monoid XXX in a functorial way. This example generalizes: any method of building a set G(X)G(X)G(X) from (the underlying set of) a monoid XXX using the monoid operations, and limits, and colimits, gives me a functor GGG.

I think it's pretty easy to show that every presheaf on TTT arises this way - in fact, just using finite products and arbitrary colimits. Every presheaf is a colimit of representables. I believe the representable presheaves on TTT come from the functors

Gn:Mon→Set G_n : \mathsf{Mon} \to \mathsf{Set} Gn​:Mon→Set

given by

Gn(X)= underlying set of Xn G_n(X) = \textrm{ underlying set of } X^n Gn​(X)= underlying set of Xn

So, we can start with just the one functor

G1(X)= underlying set of X G_1(X) = \textrm{ underlying set of } X G1​(X)= underlying set of X

and look at the presheaf on TTT that this gives:

G1∘i G_1 \circ iG1​∘i

If we then take finite products of this presheaf, and then arbitrary colimits of those products, we'll get all presheaves on TTT.

My question is: what sort of language I can use to conveniently "write down a formula" for enough GGG's to get all presheaves on TTT?

If I only use finite limits and colimits, then it seems I can write down a kind of formula for GGG using the Mitchell-Benabou language of the presheaf category on TTT. Right? Is there some generalization of this to infinite limits and colimits?

Nathanael Arkor (Apr 10 2021 at 21:26):

This may not be quite what you're asking, but the models of a Lawvere theory LLL is given by the sifted cocompletion of L°L°L°, i.e. Sind(L°)\mathrm{Sind}(L°)Sind(L°). Hence, the presheaves on LLL are given by precisely the Φ\PhiΦ-colimits of models of LLL, where Φ\PhiΦ is the class of colimits orthogonal to sifted colimits. If one could establish what colimits are given by Φ\PhiΦ, you would have a very concrete description of presheaves on Lawvere theories.
