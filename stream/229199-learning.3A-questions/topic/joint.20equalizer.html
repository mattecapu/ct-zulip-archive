David Egolf (Oct 31 2023 at 22:09):

Ah, I think I get it! We use the universal property of a limit to get our τ\tauτ:
universal property

We need s∘t=ts \circ t =t s∘t=t and t∘t=tt \circ t = tt∘t=t for the "cone" with tip GeG_eGe​ to actually be a cone. But both of these are true equations, so this is a real cone, and so we can use this setup to uniquely specify τ\tauτ.

To specify σ\sigmaσ, I think one just has to use an analogous cone with tip GeG_eGe​ with "legs" made from sss instead of ttt.
