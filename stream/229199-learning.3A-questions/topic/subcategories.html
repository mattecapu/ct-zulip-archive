John Baez (Jul 12 2021 at 21:35):

I don't understand your second version of this question, but the first version I understand, so I'll answer that.

In a category C\mathsf{C}C, the composition map

∘:hom(y,z)×hom(x,y)→hom(x,y) \circ : \mathrm{hom}(y,z) \times \mathrm{hom}(x,y) \to \mathrm{hom}(x,y) ∘:hom(y,z)×hom(x,y)→hom(x,y)

is a function between sets. So, it is not a morphism in the category C\mathsf{C}C unless C=Set\mathsf{C} = \mathsf{Set}C=Set.

There is, however, something called a closed category, which has a different kind of composition map that is actually a morphism in the category.

There is more to say about this, but my first comment - ∘\circ∘ is a function between sets, not a morphism in C\mathsf{C}C - is the most important thing to learn at first.
