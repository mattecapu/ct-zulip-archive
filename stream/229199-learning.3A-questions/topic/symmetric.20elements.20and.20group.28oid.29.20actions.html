Topic: symmetric elements and group(oid) actions

David Egolf (Dec 07 2024 at 16:59):

Let ρ:G→AutC(S)\rho:G \to \mathrm{Aut}_C(S)ρ:G→AutC​(S) be a group action, where SSS is an object of some category CCC. If we consider the case where SSS has "elements" (for example if SSS is a set or a vector space), then an element sss is "symmetric" with respect to the action if ρ(g)(s)=s\rho(g)(s) = sρ(g)(s)=s for all g∈Gg \in Gg∈G. For example, if GGG acts to rotate shapes in space about some axis, then the shapes in that space that are rotationally symmetric about that axis will be invariant under this action. As another example, if GGG acts to permute the "variable" symbols in a polynomial, then the "symmetric polynomials" are the ones invariant under this action.

I recently learned that when GGG is finite and SSS is a vector space, one can construct a "symmetric" element from any element sss of SSS. We do this by letting each element of GGG act on sss and then we add up (or "average") the results. That's really cool!

I wonder if any of these ideas can be generalized to the groupoid setting. A "groupoid action" is presumably just a functor from a groupoid GGG to some category CCC. Some questions in this direction:

  • If the objects of CCC have elements, can we obtain a reasonable notion of a "symmetric element" with respect to a groupoid action?
  • If so, can we come up with a way to "symmetrize" any element that belongs to one of the objects that GGG is acting on?

(Remember, for each g∈Gg \in Gg∈G we have ρ(g) ⁣:c→c\rho(g) \colon c \to cρ(g):c→c, so we can postcompose it with f:a→cf: a \to cf:a→c and get a new morphism ρ(g)∘f:a→c\rho(g) \circ f : a \to cρ(g)∘f:a→c. John Baez (Dec 07 2024 at 17:24):

Now, when can we 'average' an arbitrary element f:a→cf: a \to cf:a→c and get an invariant element? I know how to do it when

  1. our group is finite
  2. our category CCC is enriched over Vectk\mathsf{Vect}_kVectk​, the category of vector spaces over some field kkk with characteristic zero.

In this case we can average f:a→cf: a \to cf:a→c this way:

1∣G∣∑g∈Gρ(g)∘f \displaystyle{ \frac{1}{|G|} \sum_{g \in G} \rho(g) \circ f }∣G∣1​g∈G∑​ρ(g)∘f

and this is an invariant morphism from aaa to ccc.

John Baez (Dec 07 2024 at 17:43):

Note I need GGG to be finite for ∣G∣|G|∣G∣ to be finite, and I need a field kkk of characteristic zero to be sure that 1/∣G∣1/|G|1/∣G∣ makes sense for for any finite group GGG.

Notice that if we were working in a field of characteristic 17, like k=Z/17Zk = \mathbb{Z}/17\mathbb{Z}k=Z/17Z, then all hell would break loose in the above equation if ∣G∣|G|∣G∣ were divisible by 17.

This is actually a well-known problem in group representation theory: if we're studying the representations of a finite group over a field of characteristic ppp, it becomes a lot harder when the order of our group is divisible by ppp. Some very smart people have burnt out a lot of brain cells dealing with this.

John Baez (Dec 07 2024 at 17:45):

Finally I'll say there's a more general averaging trick we often use when GGG is not a finite group but instead a compact topological group (like the group of rotations of Rn\mathbb{R}^nRn). Essentially, "compact" is still pretty close to "finite". This makes the representation theory of compact topological groups delightfully nice compared to the noncompact ones.

John Baez (Dec 09 2024 at 21:20):

David Egolf said:

I could imagine a situation where multiple groups act on a single thing. Two group elements from different groups can't be composed, but what those group elements do can be composed.

For several groups to act on the same thing, is the same as for their coproduct to act on that thing. So, we never need to think about more than one group acting on something if we don't want to. But maybe we want to.

The coproduct of groups is, among ordinary mortals, usually called their free product, and for two groups it's written G∗HG \ast HG∗H. It's widely used when we study groups presented by generators and relations. If we take generators and relations for a group GGG and generators and relations for a group HHH and we "lump them together", we get generators and relations for G∗HG \ast HG∗H.
