Madeleine Birchfield (Mar 02 2024 at 16:24):

If RRR has decidable equality, then the set of nilradicals DDD of RRR also has decidable equality. This means that one can define any indicator function D→{0,1}⊆RD \to \{0, 1\} \subseteq RD→{0,1}⊆R by case analysis, which is a non-polynomial function if it is not the constant function to 0∈R0 \in R0∈R or 1∈R1 \in R1∈R.

Madeleine Birchfield (Mar 03 2024 at 02:20):

In general, I don't think the local requirement is necessary for the commutative ring RRR, only that RRR is non-trivial with decidable equality. If RRR is any commutative ring, then a Weil algebra WWW is equivalent to the quotient of a polynomial ring R[X]R[X]R[X] with nnn generators xi∈Xx_i \in Xxi​∈X, where 0≤i<n0 \leq i \lt n0≤i<n by a finite set of polynomials fj∈R[X]f_j \in R[X]fj​∈R[X] with cardinality mmm, where 0≤j<m0 \leq j \lt m0≤j<m. The formal spectrum SpecR(W)\mathrm{Spec}_R(W)SpecR​(W) is the solution set of the polynomials fjf_jfj​, which is a subset of RRR and thus has decidable equality if RRR has decidable equality.

Madeleine Birchfield (Mar 03 2024 at 02:40):

As a result, if RRR is non-trivial, then there are non-polynomial indicator functions SpecR(W)→{0,1}⊆R\mathrm{Spec}_R(W) \to \{0, 1\} \subseteq RSpecR​(W)→{0,1}⊆R and so the evaluation map from WWW to RSpecR(W)R^{\mathrm{Spec}_R(W)}RSpecR​(W) is not an isomorphism.
