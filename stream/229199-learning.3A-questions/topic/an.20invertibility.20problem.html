Topic: an invertibility problem

Matteo Capucci (he/him) (Mar 28 2023 at 12:27):

Suppose I have a bounded poset PPP and a functor F:P→CF:P \to \cal CF:P→C, where C\cal CC is a category. Suppose I know a≤b:Pa \leq b : Pa≤b:P, and I also know F(a≤⊤)F(a \leq \top)F(a≤⊤) is invertible. Then F(b≤⊤)F(b \leq\top)F(b≤⊤) has a right inverse, given by F(a≤⊤)−1F(a≤b)F(a \leq\top)^{-1}F(a \leq b)F(a≤⊤)−1F(a≤b).
Can I conclude F(b≤⊤)F(b \leq\top)F(b≤⊤) is left invertible too?
I'm bothered because this fact seems germane to me but it doesn't seem to be true, which means my intuition is broken. I can't concoct an example where this could go wrong.

(My actual example is a bit more structured, so maybe that's what's blinding me. In particular PPP is a fibered poset over C\cal CC and FFF is a comprehension, meaning in particular it is a right adjoint)
