pA p_A pA​ can be pulled out because of the unicity of the arrow in the definition of a pullback! fpA~ \tilde{fp_A} fpA​~​ is built out of a pullback where both arms factor through pA p_A pA​, so by unicity you get a factorization f~pA \tilde{f}p_A f~​pA​

Josselin Poiret (Jul 28 2023 at 09:10):

There's a change of POV you can make with natural models that give you equivalent definitions but much nicer reasoning imo, is instead of saying that there exists an operation λ \lambda λ that makes the square into a pullback, you can ask for an operation λ:Σ(A:U)U~A→T \lambda : \Sigma (A : U) \tilde{U}^A → T λ:Σ(A:U)U~A→T over Σ(A:U)UA \Sigma (A : U) U^A Σ(A:U)UA, where T T T is the pullback of Π Π Π and p p p, along with app:T→Σ(A:U)U~A app : T → \Sigma (A : U) \tilde{U}^A app:T→Σ(A:U)U~A. Then, asking for an equation app∘λ=id app \circ \lambda = id app∘λ=id gives Π Π Π types with only β, and if you also ask for λ∘app=id \lambda \circ app = id λ∘app=id you get η as well. I found this approach clarifies most of the reasoning here.

Josselin Poiret (Jul 28 2023 at 09:19):

also, if you have already "bootstrapped" an interpretation of dependent type theory in presheaf categories, then this is all equivalent to
⊢Ty type \vdash Ty\text{ type} ⊢Ty type
A:Ty⊢TmA type A : Ty \vdash Tm A\text{ type} A:Ty⊢TmA type (natural model).
A:Ty,B:TmA→Ty⊢piAB:Ty A : Ty, B : Tm A → Ty \vdash \mathrm{pi} A B : Ty A:Ty,B:TmA→Ty⊢piAB:Ty (type constructor)
A:Ty,B:TmA→Ty,f:(a:TmA)→Tm(Ba)⊢absf:Tm(piAB) A : Ty, B : Tm A → Ty, f : (a : Tm A) → Tm (B a) \vdash abs f : Tm (\mathrm{pi} A B) A:Ty,B:TmA→Ty,f:(a:TmA)→Tm(Ba)⊢absf:Tm(piAB) (term constructor)
A:Ty,B:TmA→Ty,f:Tm(piAB)⊢unabs:(a:TmA)→Tm(Ba) A : Ty, B : Tm A → Ty, f : Tm (\mathrm{pi} A B) \vdash unabs : (a : Tm A) → Tm (B a) A:Ty,B:TmA→Ty,f:Tm(piAB)⊢unabs:(a:TmA)→Tm(Ba) (term "reifier", it's not really an eliminator)
A:Ty,B:TmA→Ty,f:(a:TmA)→Tm(Ba)⊢unabs(absf)=f A : Ty, B : Tm A → Ty, f : (a : Tm A) → Tm (B a) \vdash unabs (abs f) = f A:Ty,B:TmA→Ty,f:(a:TmA)→Tm(Ba)⊢unabs(absf)=f (β)
A:Ty,B:TmA→Ty,f:Tm(piAB)⊢abs(unabsf)=f A : Ty, B : Tm A → Ty, f : Tm (\mathrm{pi} A B) \vdash abs (unabs f) = f A:Ty,B:TmA→Ty,f:Tm(piAB)⊢abs(unabsf)=f (η)

This was very illuminating to me! It's the basic idea behind 2LTT and variations!
