Topic: Categories generalizing groupoids and preorders

John Baez (Jan 16 2026 at 00:48):

Has anyone written about categories with this property?

Property A. Given morphisms f:x→yf: x \to yf:x→y and g:y→xg: y \to xg:y→x, fff must be the inverse of ggg.

Note that any preorder has this property, and so does any groupoid.

More generally, I believe we can get a lot of categories with property A as follows. Take any preorder PPP and any pseudofunctor F:P→GpdF: P \to \mathbf{Gpd}F:P→Gpd, and do the covariant Grothendieck construction ∫F\int F∫F. Then I believe ∫F\int F∫F has property A. The category ∫F\int F∫F is like a "bundle of groupoids over a preorder".

But I believe it's not true that any category with property A is given by the above construction! A counterexample would a category with only two objects xxx and yyy, two morphisms f,g ⁣:x→yf, g \colon x \to yf,g:x→y, and identity morphisms for xxx and yyy.

This category is not a preorder, but it has a weaker property shared by all preorders:

Property B. If there is a morphism f:x→yf : x \to yf:x→y that is not an isomorphism, there is no morphism g:x→yg: x \to yg:x→y.

Has anyone studied categories with property B?
