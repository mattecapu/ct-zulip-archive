Put Y=0Y = 0Y=0 and Z=1Z = 1Z=1 in the natural isomorphism above:
1≅0+1≅0×1≅01 \cong 0 + 1 \cong 0 \times 1 \cong 01≅0+1≅0×1≅0

Ralph Sarkis (May 28 2025 at 14:45):

The terminal object is always the unit of the product: X×1≅XX \times 1 \cong XX×1≅X for any XXX. Dually, the initial object is always the unit of the coproduct: X+0≅AX + 0 \cong AX+0≅A for any XXX. The first sentence of the proof in the screenshot seems to make this more complicated than it is.
