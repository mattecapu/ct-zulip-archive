Topic: faithfulness and Cauchy completions

John Baez (Jul 29 2024 at 14:39):

In a paper I'm coauthoring we want to use the following fact, and we'd like a reference for it:

If CCC and DDD are kkk-linear categories, F:C→DF : C \to DF:C→D is a fully faithful kkk-linear functor, and DDD is Cauchy complete, then the functor C‾→D\overline{C} \to DC→D obtained by extending FFF to the Cauchy completion C‾\overline{C}C is also fully faithful.

John Baez (Jul 29 2024 at 14:40):

Does anyone know a reference for this, or a reference for some more general fact of which this quickly falls out as a consequence?

Nathanael Arkor (Jul 29 2024 at 16:33):

The more general fact is that if Φ\PhiΦ is a class of weights, then a functor f:A→Bf : A \to Bf:A→B is fully faithful if and only if the induced functor Φ(f):Φ(A)→Φ(B)\Phi(f) : \Phi(A) \to \Phi(B)Φ(f):Φ(A)→Φ(B) between their Φ\PhiΦ-cocompletions is fully faithful.

I don't actually know an explicit reference for this, but it is an easy consequence of the fact that if f:A→Bf : A \to Bf:A→B is fully faithful, and g,g′:A→Cg, g' : A \to Cg,g′:A→C are functors, then C(Lanfg,Lanfg′)≅C(g,g′)C(\text{Lan}_f g, \text{Lan}_f g') \cong C(g, g')C(Lanf​g,Lanf​g′)≅C(g,g′), assuming the pointwise left extensions exist.
