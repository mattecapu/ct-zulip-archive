Topic: small products in model categories

Elif Uskuplu (Mar 28 2023 at 19:21):

Hi,
I don't know much about model categories, but as I've been working from Hovey's book, I'd like to ask a question. As in the attached file, Screenshot-from-2023-03-28-12-00-14.png we have the homotopy category of a model category has all small products.

From this, I can talk about countable products in particular. Well, can we say the following?
If the maps fn,gn:B→Xnf_n,g_n:B\rightarrow X_nfn​,gn​:B→Xn​ are left homotopic for each n∈Nn\in\mathbb{N}n∈N, and f=∏fnf=\prod f_nf=∏fn​ and g=∏gng=\prod g_ng=∏gn​, then f,g:B→∏Xnf,g:B\rightarrow \prod X_nf,g:B→∏Xn​ are left homotopic. It seems right to me, but I'm not sure about the details or extra assumptions, if needed. Any explanation or correction would be great.

Mike Shulman (Mar 29 2023 at 04:12):

This is true if BBB is cofibrant and each XnX_nXn​ is fibrant. In that case, there's a lemma saying that if fn∼gnf_n \sim g_nfn​∼gn​ then that left homotopy can be witnessed by any desired cylinder object for BBB. So if you pick a single cylinder object B~\tilde{B}B~ for BBB and use it for all the homotopies fn∼gnf_n \sim g_nfn​∼gn​, then you can induce a map B~→∏nXn\tilde{B} \to \prod_n X_nB~→∏n​Xn​ and deduce f∼gf\sim gf∼g.

Reid Barton (Mar 29 2023 at 06:59):

And, it's not true in general. For instance in simplicial sets, if BBB is two points and XnX_nXn​ is a path/string of 1-simplices of length nnn, and fnf_nfn​ picks the first vertex of the each path and gng_ngn​ the last vertex. Then fn∼gnf_n \sim g_nfn​∼gn​ for each nnn but in the product fff and ggg are actually points in two different connected components, basically because any path in the product has finite length so it's too short to connect the endpoints after projecting to XnX_nXn​ for large nnn.

Reid Barton (Mar 29 2023 at 07:01):

So you have to do a fibrant replacement before taking infinite products, to get a homotopy invariant construction.
In sSet you don't have to do fibrant replacement to form "finite homotopy products", but in a general model category you could have to.
