Topic: Implication of walking isomorphism ≅ single object?

Madeleine Birchfield (Mar 16 2025 at 18:36):

Ryo Suzuki said:

It might be related to the notion of path induction in HoTT, but I don’t know much about it…

In HoTT, there is the unit type 111 and the interval type I\mathbb{I}I. Functions from 111 to a type AAA correspond to elements in AAA, while functions from I\mathbb{I}I to AAA correspond to paths / identifications / equalities in AAA, which are always invertible by path induction. One can prove that 111 and I\mathbb{I}I are both singletons (i.e. contractible types) and thus equivalent to each other

If we interpret types AAA as ∞\infty∞-groupoids, then elements of types are objects in ∞\infty∞-groupoids, while paths of types are 1-morphisms of ∞\infty∞-groupoids, which are always isomorphisms by definition of ∞\infty∞-groupoid. 111 is the terminal ∞\infty∞-groupoid, while I\mathbb{I}I is the walking isomorphism.
