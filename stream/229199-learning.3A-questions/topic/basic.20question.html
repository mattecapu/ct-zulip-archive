well, a (-1)-category is a truth value, an element of {⊤,⊥}\{\top,\perp\}{⊤,⊥}. If you want to keep having a sequence of inclusions (−1)-Cat⊆0-Cat⊆1-Cat⊆…(-1)\text{-Cat} \subseteq 0\text{-Cat} \subseteq 1\text{-Cat} \subseteq \dots(−1)-Cat⊆0-Cat⊆1-Cat⊆… then you can represent (−1)-Cat(-1)\text{-Cat}(−1)-Cat as the subcategory of SetSetSet spanned by the empty set and a singleton.

There aren't many ways in which you can equip one of these with a monoid operation...
