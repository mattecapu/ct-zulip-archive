Topic: Confusion with conservative adjoints

John Onstead (Jun 24 2025 at 11:16):

There's a claim that if a functor R:C→DR: C \to DR:C→D has a left adjoint and CCC is finitely complete, RRR is conservative if and only if every counit component is strong epi. Interestingly, this implies that if CCC is finitely complete, any conservative functor with a left adjoint must be faithful, since being faithful in an adjunction is the same counit requirement but without the "strongness" in the epis.
