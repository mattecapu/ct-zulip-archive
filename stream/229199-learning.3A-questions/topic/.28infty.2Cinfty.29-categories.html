Topic: (infty,infty)-categories

Leopold Schlicht (Feb 28 2022 at 15:50):

There is a subclass of the class of all simplicial sets which provides a model for (∞,1)(\infty,1)(∞,1)-categories: the simplicial sets in which every inner horn has a filler ("weak Kan extension condition"). I just noticed that there is also a (larger) subclass of simplicial sets which provides a model for (∞,2)(\infty,2)(∞,2)-categories: Kerodon 01W9.

Can (∞,∞)(\infty,\infty)(∞,∞)-categories be defined as certain simplicial sets too? Is there any literature about that?
