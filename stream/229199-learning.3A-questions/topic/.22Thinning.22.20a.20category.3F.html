Topic: "Thinning" a category?

Shea Levy (Oct 14 2020 at 11:27):

Is there a name for the process of generating a new category from some arbitrary category by keeping the objects the same but identifying all the morphisms between the same objects? "Thinning" maybe?

Matteo Capucci (he/him) (Oct 14 2020 at 11:30):

It's changing base of enrichment, from Set\mathbf{Set}Set to Bool={0<1}\mathbf{Bool} = \{0 < 1\}Bool={0<1}. Notice there is a canonical arrow from the first to the latter, which sends ∅\varnothing∅ in 000 and everything else in 111, and you can figure out what it does on arrows. Then the change of base along this functor is what you're after.

sarahzrf (Oct 14 2020 at 14:41):

fun things to note: if we identify Ω with the subcategory of Set on subsingleton sets (this is [almost] precisely analogous to identifying Set with the subcategory of Cat on discrete categories—(-1)-Cat into 0-Cat instead of 0-Cat into 1-Cat), then the functor along which we change base of enrichment is in fact the left adjoint to the inclusion

sarahzrf (Oct 14 2020 at 14:49):

more broadly:

  • an object whose yoneda embedding is a functor valued in that subcategory of Set is gonna be precisely a subterminal object
  • given some Set-flavored category, it's of interest to ask whether the full subcategory of subterminals—which is always a preorder—is reflective, as it is in Set, because that will be a kind of "propositional truncation"
  • given a category with finite limits, it's further of interest to ask whether it locally admits that, in the sense of "slices have the thing and pullback respects it"—i.e., whether we have reflective subterminals in each slice category, and the base change functors commute with the reflections. that's exactly a regular category

sarahzrf (Oct 14 2020 at 14:51):

...subterminals in C/A are exactly monomorphisms into A, and it turns out that reflecting a random object of C/A into a subterminal gives you an image factorization (the reflection unit is the other part of the factorization)—this is like saying that you're "truncating each fiber to a subsingleton so that you get an injection"
