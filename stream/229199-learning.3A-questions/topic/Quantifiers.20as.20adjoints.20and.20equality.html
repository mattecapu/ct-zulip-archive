Topic: Quantifiers as adjoints and equality

Jencel Panic (Nov 12 2023 at 09:52):

This is a spinoff to the discussion we have on "Beginner questions" on the topic of quantifiers as adjoints. I want to point out some connections that I see between the adjunctions and categorical equivalences, particularily when it comes to quantifiers. I am doing it in a separate topic, because I am not sure if I am correct and I don't want to confuse people.

My source for this is example 7.1.1.13 from Spivak's "Category theory for the sciences", namely the fact that given a function $p: A \to B$ the inverse image of this function p−1:P(B)→P(A)p^{-1} : P(B) \to P(A)p−1:P(B)→P(A) is left adjoint of the ∀p:P(A)→P(B)\forall p : P(A) \to P(B) ∀p:P(A)→P(B) and right adjoint of ∃p:P(A)→P(B)\exists p : P(A) \to P(B) ∃p:P(A)→P(B).

image.png

Now, if I got the picture right, one thing I am noticing is that besides adjunctions, these pairs of functors define a "half-equivalence" or "one-direction equivalence" (I swear there was a name for this, but I don't remember it). So, on one hand we have ∀p∘p−1=IDB\forall p \circ p^{-1}= ID_{B} ∀p∘p−1=IDB​ (but we don't have the other direction p−1∘∀p≠IDA p^{-1} \circ \forall p \neq ID_{A} p−1∘∀p=IDA​). And same for exists --- we have p−1∘∃p=IDBp^{-1} \circ \exist p = ID_{B} p−1∘∃p=IDB​, but not the other direction.

Moreover, in the cases where the function ppp is an isomorphism, this partial equivalence becomes full equivalence (I think that's not the only case, but I am not sure).

The same things can happen if we remove some elements from P(A)P(A)P(A).

Here is a visualization of the example in the book --- the adjunction between the inclusion order of the natural numbers and the inclusion order of the set even,odd{even, odd}even,odd, that is generated by the function isEven:N→BoolisEven : \mathbb{N} \to BoolisEven:N→Bool.

Todd Trimble (Nov 12 2023 at 12:22):

It's even better to refer to specific morphisms as isomorphisms or not. So instead of wondering whether, for example, p−1∘∃pp^{-1} \circ \exists pp−1∘∃p is the identity, the issue should be whether the unit of the adjunction ∃p⊣p−1\exists p \dashv p^{-1}∃p⊣p−1, which is of the form

1B→p−1∘∃p,1_B \to p^{-1} \circ \exists p,1B​→p−1∘∃p,

is an isomorphism. Of course if your categories are posets, this is asking the same thing since there is at most one morphism between these functors to begin with. But in general, it's better to ask the second question.

Todd Trimble (Nov 12 2023 at 12:23):

This is a great example to work with when learning adjoints, because the functors here are very familiar constructions. ∃p:PA→PB\exists p: PA \to PB∃p:PA→PB takes a subset XXX of AAA to its direct image p(X)p(X)p(X), so the statement that we have an adjunction ∃p⊣p−1\exists p \dashv p^{-1}∃p⊣p−1 is that for subsets X,YX, YX,Y of A,BA, BA,B respectively, p(X)⊆Yp(X) \subseteq Yp(X)⊆Y if and only if X⊆p−1(Y)X \subseteq p^{-1}(Y)X⊆p−1(Y). The unit assigns to each such XXX a subset inclusion X⊆p−1(p(X))X \subseteq p^{-1}(p(X))X⊆p−1(p(X)), and the counit assigns to each such YYY a subset inclusion p(p−1(Y))⊆Yp(p^{-1}(Y)) \subseteq Yp(p−1(Y))⊆Y.

Todd Trimble (Nov 12 2023 at 12:23):

Here, the unit is an isomorphism iff ppp is injective. For example, consider a noninjective function like the function p:{a,b}→{1}p: \{a, b\} \to \{1\}p:{a,b}→{1}. Notice that p−1(p({a}))=p−1({1})={a,b}p^{-1}(p(\{a\})) = p^{-1}(\{1\}) = \{a, b\}p−1(p({a}))=p−1({1})={a,b}, so the unit component {a}⊆(p−1∘p)({a})\{a\} \subseteq (p^{-1} \circ p)(\{a\}){a}⊆(p−1∘p)({a}) is not an isomorphism. Incidentally, in general when the unit is an isomorphism, the left adjoint part is called a coreflection.

Todd Trimble (Nov 12 2023 at 12:23):

There is also a nice exercise that the counit of ∃p⊣p−1\exists p \dashv p^{-1}∃p⊣p−1 is an isomorphism iff ppp is surjective.
