To explain a little more, we know there is a free functor

F:Alg→DiffAlg.F: \mathsf{Alg} \to \mathsf{DiffAlg}.F:Alg→DiffAlg.

A differential algebra freely constructed this way is "universal", i.e.

F(A)=(A,Ω~1,d~:A→Ω~1).F(A) = (A,\tilde\Omega^1,\tilde d: A\to\tilde\Omega^1).F(A)=(A,Ω~1,d~:A→Ω~1).

We also know (since d~:A→Ω~1\tilde d: A\to\tilde\Omega^1d~:A→Ω~1 is universal) that any other differential algebra (A,Ω1,d:A→Ω1)(A,\Omega^1,d: A\to\Omega^1)(A,Ω1,d:A→Ω1) factors through this one and there is a unique (A,A)(A,A)(A,A)-bimodule morphism

ϕ:Ω~1→Ω1\phi: \tilde\Omega^1\to\Omega^1ϕ:Ω~1→Ω1

with

d=ϕ∘d~.d = \phi \circ \tilde d.d=ϕ∘d~.

If AAA is generated from the vertices of a directed graph, then the (A,A)(A,A)(A,A)-bimodule morphism ϕ\phiϕ is generated from the "adjacency matrix" of the directed graph we call the "graph operator" G⊂A⊗KAG \subset A\otimes_K AG⊂A⊗K​A given by

G=∑i,jG(i,j)ei⊗KejG = \sum_{i,j} G(i,j) e^i\otimes_K e^jG=∑i,j​G(i,j)ei⊗K​ej

where G(i,j)=1G(i,j) = 1G(i,j)=1 if there is a directed edge i→ji\to ji→j and zero otherwise.

So to summarize, if I only give you a set of vertices, I can construct a "universal differential algebra" on that set which corresponds to the complete graph. To get another differential algebra besides the universal one, we need to set some of the directed edges to zero. That is captured by GGG so we need the full data of a directed graph to construct a general differential algebra.

I'm writing the linear map as ddd just to make you interested in it - in general it's just any linear map. Similarly, the phrase "short chain complex" is supposed to give you ideas about why we care about this, but it's just a name for two vector spaces and a linear map!

  • morphisms are natural transformations.
