Smooth brain moment: is the following true for a function fff?

Suppose fff is invertible iff fff is surjective. Then fff is injective.

Todd Trimble (Aug 26 2024 at 16:33):

It would help to put quantifiers in as appropriate. I can say that if AAA is a finite set, then it is unconditionally true that for all f:A→Af: A \to Af:A→A, fff is surjective iff fff is invertible. That doesn't mean that f:A→Af: A \to Af:A→A is always injective. But perhaps I misunderstood what is being asked.
