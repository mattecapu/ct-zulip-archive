Topic: Minimal set of axioms to make affine logic intuitionistic

Madeleine Birchfield (Jan 13 2025 at 16:31):

Is there a minimal set of axioms or inference rules that one can add directly to affine logic to make it equivalent to an antithesis translation of intuitionistic logic into affine logic?
