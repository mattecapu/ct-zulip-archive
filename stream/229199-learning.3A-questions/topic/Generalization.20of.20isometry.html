Topic: Generalization of isometry

Sascha Haupt (May 08 2025 at 10:45):

I am looking for a name for the following generalization of the concept of isomtery in a dagger category:
In a [[dagger category]], an isomtery is a morphism A→BA\rightarrow BA→B is called an isometry if f†∘f=idAf^{\dagger}\circ f = id_Af†∘f=idA​ and a partial isometry when f†∘ff^{\dagger}\circ ff†∘f is dagger idempotent.
Is there a name for a morphism fff with the property that f†∘ff^{\dagger}\circ ff†∘f is an isomorphism?

Amar Hadzihasanovic (May 08 2025 at 11:25):

Just some comments as to “why maybe there isn't a name already”:
typically in dagger-categories it is “wrong” to talk about isomorphisms which are not unitary isomorphisms, so the more likely concept is: fff such that f†∘ff^\dagger \circ ff†∘f is unitary.
Now, f†∘ff^\dagger \circ ff†∘f is also self-adjoint, so in particular being unitary would mean it is an involution.
Both in Rel\mathbf{Rel}Rel and in Hilb\mathbf{Hilb}Hilb there are no examples in which fff is not already an isometry:

  • in Rel\mathbf{Rel}Rel because for R†∘RR^\dagger \circ RR†∘R to be invertible, RRR would need to be total, in which case R†∘RR^\dagger \circ RR†∘R always contains the identity relation, and so in order to be an involution, it has to be the identity relation;
  • in Hilb\mathbf{Hilb}Hilb because A†AA^\dagger AA†A is positive, so its spectrum is positive, but an involution must have spectrum in {+1,−1}\{+1, -1\}{+1,−1}, so a positive involution has spectrum {+1}\{+1\}{+1} and then it is the identity

Amar Hadzihasanovic (May 08 2025 at 11:28):

Now I think pretty much any “named concept” in dagger category theory is an extrapolation from Rel\mathbf{Rel}Rel and Hilb\mathbf{Hilb}Hilb, so if there are no examples in these two I see it as unlikely that it would have shown up!

Amar Hadzihasanovic (May 08 2025 at 11:29):

But there seems to be no strong reason for this to happen in general... Do you have an example in mind?

Aaron David Fairbanks (May 08 2025 at 15:53):

It is called a "closed monomorphism" by Matt Di Meglio in R*-categories, because in Hilb\mathbf{Hilb}Hilb it corresponds to a monomorphism whose image is closed. It's also equivalent to being monic and pseudoinvertible (a fact that appears in the extended version of my paper about pseudoinverses now available here).
