John Baez (Mar 27 2020 at 20:18):

Let's see how I'd prove the result in Aluffi. I think this is more pleasant if we introduce some terminology and concepts. Let me use additive notation for group operations since all the groups involved must be abelian: since Q\mathbb{Q}Q is, so must be GGG and HHH.

Given a group element aaa let's call aaa added to itself nnn times for n=1,2,3...n = 1, 2, 3...n=1,2,3... a multiple of aaa.

A group is torsion-free if when any multiple of an element is zero, that element must be zero.

Q\mathbb{Q}Q is torsion-free. Any subgroup of a torsion-free group is torsion-free.

Now assume Q≅G×H\mathbb{Q} \cong G \times HQ≅G×H. We wish to show either GGG or HHH is trivial. To do this let's assume HHH is nontrivial and show GGG is trivial.

Since GGG and HHH are isomorphic to subgroups of Q\mathbb{Q}Q they must be torsion-free.

Let πG ⁣:Q→G\pi_G \colon \mathbb{Q} \to G πG​:Q→G be the projection onto GGG.

Since HHH is nontrivial there must be some nonzero element q∈Qq \in \mathbb{Q}q∈Q with πG(q)=0\pi_G(q) = 0πG​(q)=0. This implies that πG\pi_GπG​ of any multiple of qqq is also zero. But for every q′∈Qq' \in \mathbb{Q}q′∈Q we can find some multiple of q′q'q′ that is also a multiple of qqq. So, for every q′∈Qq' \in \mathbb{Q}q′∈Q, πG\pi_GπG​ of some multiple of q′q'q′ is zero. Since GGG is torsion-free this implies that πG(q′)=0\pi_G(q') = 0πG​(q′)=0.

In other words, πG\pi_GπG​ maps everything to zero. So GGG must be trivial!
