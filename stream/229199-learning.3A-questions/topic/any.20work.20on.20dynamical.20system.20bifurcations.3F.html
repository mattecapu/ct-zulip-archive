Maybe? One can surely arrange a dynamical system as a lens (S,S)⇆(1,1)(S, S) \leftrightarrows (1,1)(S,S)⇆(1,1). This is a map of polynomials δ:SyS→y\delta:Sy^S \to yδ:SyS→y.
If your dynamics is parameterized by PPP, then it is a map of polynomials P⊗SyS→yP \otimes Sy^S \to yP⊗SyS→y. But this is merely repackaging the basic definitions. What can the categorical framework say about the bifurcation diagram?
