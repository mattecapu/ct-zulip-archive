Topic: weighted colimit is an evaluation

fosco (Sep 28 2020 at 10:23):

This should be well-known, but I can't find a reference.

Given a functor F:C→DF : {\cal C} \to {\cal D}F:C→D, if C\cal CC has a terminal object ttt, colim F\text{colim } Fcolim F coincides with the evaluation of FFF at ttt.

What about weighted colimits? Let's say F:C→DF : {\cal C} \to {\cal D}F:C→D is a V\cal VV-functor and W:Cop→VW:{\cal C}^{op} \to {\cal V}W:Cop→V a weight. If V=Set{\cal V}={\sf Set}V=Set it's easy to see that the following conditions are equivalent:

  • the category of elements of WWW has a terminal object
  • WWW is a representable functor, say hom⁡(−,A)\hom(-,A)hom(−,A).

Thus, colimWF≅FA\text{colim}^W F\cong FAcolimWF≅FA.

But what if there is no category of elements construction for V\cal VV?

Martti Karvonen (Sep 28 2020 at 14:51):

The fact about terminal objects corresponds to the inclusion of the terminal object into C\mathcal{C}C being a final functor. I think Kelly discussess final functors in passing (and has some results about weighted colimits too) in section 4.5. in the enriched cats book - perhaps this will satisfy your needs.

fosco (Sep 28 2020 at 17:43):

Ah, yes, it is a good idea to rephrase the property into a finality request. Thank you!
