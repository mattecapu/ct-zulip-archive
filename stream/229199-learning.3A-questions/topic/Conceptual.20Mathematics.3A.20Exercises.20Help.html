Ralph Sarkis (May 22 2021 at 08:59):

The elements in the left hand side are functions f:N→Yf: \mathbb{N} \rightarrow Yf:N→Y making the square below commute.
image.png

The elements in the right hand side are elements y∈Yy \in Yy∈Y viewed as functions y:1→Yy: \mathbf{1} \rightarrow Yy:1→Y.

The goal of the exercise is to show that the elements in the LHS are in correspondence with the elements in the RHS. Starting with f:N→Yf: \mathbb{N} \rightarrow Yf:N→Y in the LHS, there is a straightforward way to obtain an element of YYY, apply fff to 0∈N0 \in \mathbb{N}0∈N to obtain f(0)∈Yf(0) \in Yf(0)∈Y, this is "evaluation at 0".

The other direction is less straightforward. First, we observe that any f:N→Yf: \mathbb{N} \rightarrow Yf:N→Y in the LHS is completely determined by f(0)f(0)f(0). Indeed, by the commutativity of the square above, you can derive the value of f(1)f(1)f(1) by f(1)=f(σ(0))=β(f(0))f(1) = f(\sigma(0)) = \beta(f(0))f(1)=f(σ(0))=β(f(0)), the value of f(2)f(2)f(2) by f(2)=f(σ(1))=β(f(1))f(2) = f(\sigma(1)) = \beta(f(1))f(2)=f(σ(1))=β(f(1)) and so on. More concisely, you can define f(n)=βn(f(0))f(n) = \beta^n(f(0))f(n)=βn(f(0)) where βn\beta^nβn denotes the iteration of β\betaβ nnn times, i.e.: βn=β∘⋯n∘β\beta^n = \beta \circ \stackrel{n}{\cdots} \circ \betaβn=β∘⋯n∘β. Now, if you are given y∈Yy \in Yy∈Y, you can set f(0)=yf(0) = yf(0)=y and iterate β\betaβ to find all the other values of fff. This yields the direction LHS -> RHS.

Finally, the exercise asks you to show the two operation we just described are inverses. Namely, if you start with y∈Yy \in Yy∈Y, you define fff as above and you evaluate it at 0, you should obtain yyy. Conversely, if you start with ggg in the LHS and you evaluate at 0 and define fff as above setting f(0)=g(0)f(0) = g(0)f(0)=g(0) you should conclude f=gf = gf=g.
