Interesting stuff! One way to see why x+1=1x+1=1x+1=1 is so important might be as follows. If we set x≤yx \leq yx≤y iff xy=xxy=xxy=x (this is the ordering we get from multiplication), then in particular x≤1x \leq 1x≤1 for all xxx, because 111 is the multiplicative unit.
