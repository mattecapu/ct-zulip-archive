John Baez (May 01 2021 at 14:24):

Okay, that definition looks good. For category theorists a graph (also called a directed multigraph or quiver) is a pair of functions s,t:E→Vs, t: E \to Vs,t:E→V. Taking the pullback of these we get the set of length-two paths, and this pullback has two maps to EEE. So, we get a new graph, the line graph, where the set of vertices is EEE and the set of edges is the set of length-two paths.

John Baez (May 01 2021 at 14:27):

So yes, one question is whether there's a version of Whitney's isomorphism theorem for quivers: the original version is for graph theorists' graphs (also called simple graphs).

Another question we could ask is whether the line-graph functor is an adjoint.

It's definitely not a right adjoint since it doesn't preserve the terminal object (the point gets sent to the empty graph).

If it's a left adjoint, with right adjoint UUU, then we would have VU(G)=Hom(1,U(G))=Hom(F(1),G)=Hom(0,G)=1V_{U(G)} = \mathrm{Hom}(1,U(G)) = \mathrm{Hom}(F(1),G) = \mathrm{Hom}(0,G) = 1VU(G)​=Hom(1,U(G))=Hom(F(1),G)=Hom(0,G)=1 and EU(G)=Hom(∙→∙,U(G))=Hom(F(∙→∙),G)=Hom(1,G)=VGE_{U(G)} = \mathrm{Hom}(\bullet\to\bullet,U(G)) = \mathrm{Hom}(F(\bullet\to\bullet),G) = \mathrm{Hom}(1,G) = V_GEU(G)​=Hom(∙→∙,U(G))=Hom(F(∙→∙),G)=Hom(1,G)=VG​. So then maps H→U(G)H\to U(G)H→U(G) would just be arbitrary functions EH→VGE_H\to V_GEH​→VG​, which is very different from F(H)→GF(H)\to GF(H)→G.

Now I think there is a right adjoint.
