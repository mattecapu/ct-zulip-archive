Topic: a simple 2-categorical exercise

Ambroise (Oct 11 2023 at 16:53):

Hello, I am trying to improve my 2-categorical skills with the following exercise: prove that if a category B has an initial object b, then a natural transformation α\alphaα between a constant functor from B and another one is uniquely determined by its component αb\alpha_bαb​. This is easy to show directly, but is there a 2-categorical proof of that relying on some adequate notions of an object of a 2-category with an initial object and similar stuff?

Patrick Nicodemus (Oct 11 2023 at 19:30):

I don't know how standard this is, but I have recently been working with 2-categories a lot, and a useful definition for my purposes is
that if A\mathcal{A}A is a given 2-category and XXX is an object in A\mathcal{A}A, XXX has an initial object if the category A(Y,X)\mathcal{A}(Y,X)A(Y,X) has an initial object for every YYY, and these initial objects are preserved under the functors A(Y,X)→A(Z,X)\mathcal{A}(Y,X)\to\mathcal{A}(Z,X)A(Y,X)→A(Z,X) given by reindexing. In some cases it might be useful to use the stronger definition that there is a specific choice of terminal object in each category A(Y,X)\mathcal{A}(Y,X)A(Y,X) and these specific choices are preserved under reindexing. Such a specific choice of section would correspondto the 2-presheaf A(−,X)\mathcal{A}(-,X)A(−,X) having a "global section" from the terminal presheaf 1:A→Cat\mathbf{1}:\mathcal{A}\to\mathbf{Cat}1:A→Cat which sends everything to the terminal category.

Kengo Hirata (Oct 12 2023 at 15:09):

I have a 2-categorical proof with a 2-terminal object 1.
First, observe that a category with an initial object is an object BBB in Cat\mathbf{Cat}Cat with an adjunction b⊣!:1→Bb\dashv ! : 1 \rightarrow Bb⊣!:1→B.
Then, from the mate correspondence, a 2-cell b∘!⇒fb \circ ! \Rightarrow fb∘!⇒f is bijective to !⇒!∘f! \Rightarrow !\circ f!⇒!∘f which is unique by the definition of terminal object.

Patrick Nicodemus (Oct 12 2023 at 15:15):

What I meant is the following. I don't insist this is the unique or most natural way to adapt this to 2-categories, just what came to mind.

Let Y,XY,XY,X be objects in our 2-category A\mathcal{A}A. Let a,b:Y→Xa, b : Y\to Xa,b:Y→X and τ:a⇒b\tau :a\Rightarrow bτ:a⇒b. Assume A\mathcal{A}A has a 2-terminal object 1\mathbf{1}1 and that aaa and bbb both factor through the 2-categorical object by 1-cells a′,b′:1→Xa', b' : \mathbf{1}\to Xa′,b′:1→X.

Now let ZZZ be arbitrary and j:Z→Yj : Z\to Yj:Z→Y. We have an initial object 0∈A(Z,Y)0 \in \mathcal{A}(Z,Y)0∈A(Z,Y) and an initial map e:0⇒je : 0\Rightarrow je:0⇒j.
By some basic theorems about the naturality of whiskering, we have that b⋅e∘τ⋅0:a⋅0⇒b⋅0⇒bj=τ⋅j∘a⋅e:a⋅0⇒a⋅j⇒b⋅jb\cdot e\circ \tau\cdot 0 : a\cdot 0\Rightarrow b\cdot 0\Rightarrow bj = \tau\cdot j\circ a\cdot e : a\cdot 0\Rightarrow a\cdot j\Rightarrow b\cdot jb⋅e∘τ⋅0:a⋅0⇒b⋅0⇒bj=τ⋅j∘a⋅e:a⋅0⇒a⋅j⇒b⋅j.

Because a,ba,ba,b factor through the terminal object, a⋅j=a⋅0a\cdot j =a\cdot 0a⋅j=a⋅0 and a⋅ea\cdot ea⋅e is the identity 2-cell, and similarly b⋅0⇒b⋅jb\cdot 0\Rightarrow b\cdot jb⋅0⇒b⋅j is the identity 2-cell. This forces τ⋅0=τ⋅j\tau\cdot 0 = \tau\cdot jτ⋅0=τ⋅j. Thus τ⋅j\tau\cdot jτ⋅j is determined by τ⋅0\tau\cdot 0τ⋅0.
The intuition here is that τ⋅j\tau\cdot jτ⋅j is the component of τ\tauτ on the "generalized object" jjj. One can always take jjj to be the identity 1-cell on YYY, and then you get a particularly strong instance of this claim, that τ\tauτ itself is completely determined by τ⋅0\tau\cdot 0τ⋅0.
