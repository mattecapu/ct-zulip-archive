Ryan Wisnesky (Jul 28 2024 at 23:22):

one answer to this question is that a cartesian closed category (such as the simply typed lambda calculus) allows higher order programming, whereas a cartesian category (such as equational logic) only allows first order programming. for many purposes this is not a meaningful distinction - but for many purposes it is meaningful. For a technical answer to your question, you can define exponentiation with an object X as adjoint to taking a product with X, and probe exponentiation that way.

John Baez (Jul 30 2024 at 13:57):

Another answer: there's

  • a set of maps from a set to a set,
  • a poset of order-preserving maps from a poset to a poset,
  • a category of functors from a category to a category,
  • a vector space of linear maps from a vector space to a vector space,
  • a topological space of continuous maps from a topological space to a topological space,

etc., and we'd like to formalize this pattern:

there's a thing (of some sort) of maps from any thing (of that sort) to any other thing (of that sort)

Formalizing this pattern requires [[cartesian closed categories]] but also more general [[monoidal closed categories]].

John Baez (Aug 01 2024 at 13:57):

It doesn't pay to get fancy and bring model theory into this conversation before you get the basic idea, which is pretty simple.

There's no way you can look at a set SSS and tell whether it's a group or not, because a group is a set SSS with extra structure: an operation ⋅\cdot⋅ and an element 111 obeying some equations. You can ask if the triple (S,⋅,1)(S,\cdot, 1)(S,⋅,1) is a group or not. It makes no sense to ask if SSS is a group or not.

Similarly, there's no way you can look at a category MMM and tell whether it's monoidal or not, because a monoidal category is a category MMM with some extra structure: a functor ⊗\otimes⊗ and an object III and some natural transformations a,ℓ,ra, \ell, ra,ℓ,r obeying some equations. You can ask if the sextuple (M,⊗,I,a,ℓ,r)(M, \otimes, I, a, \ell, r)(M,⊗,I,a,ℓ,r) is a monoidal category or not. It makes no sense to ask if MMM is a monoidal category or not.

I would say the motivation is that it is a fairly minimal hypothesis that suffices to get the proof of the Lawvere fixed point theorem to work. "Weak point surjectivity" is not, I don't think, an especially important concept on its own.

Let's just look at the proof a moment, as played out in the category of sets. The statement is that if there is an endofunction f:B→Bf: B \to Bf:B→B with no fixed point b:1→Bb: 1 \to Bb:1→B, then there can be no surjective map A→BAA \to B^AA→BA for any set AAA. Or, to phrase it more positively: if there exists a surjective map ϕ:A→BA\phi: A \to B^Aϕ:A→BA, then every function f:B→Bf: B \to Bf:B→B has a fixed point. The proof proceeds as follows:

  1. Let q:1→BAq: 1 \to B^Aq:1→BA name the function A→BA \to BA→B that sends an element a∈Aa \in Aa∈A to f(ϕ(a)(a))f(\phi(a)(a))f(ϕ(a)(a)).   2. Claim: ϕ(a0)(a0)\phi(a_0)(a_0)ϕ(a0​)(a0​) is a fixed point of fff. This is because
    ϕ(a0)(a0)=q(a0)=f(ϕ(a0)(a0))\phi(a_0)(a_0) = q(a_0) = f(\phi(a_0)(a_0))ϕ(a0​)(a0​)=q(a0​)=f(ϕ(a0​)(a0​))
    according to how qqq was defined, QED.

This simple proof can be abstracted and generalized so that it holds in categories much more general than the category of sets. Believe it or not, it's the other way around. If you say that someone is making too strong a claim, like "I am the best educated American", that means you think it's unlikely to be true. Conversely, a weaker statement like "I had an above average education for an American" is more likely to be true. Stronger statements imply relatively weaker statements. If I am the best educated American, then it would follow that I had an above average education for an American.

The weakest statements are the tautologies, the unconditional truisms, the things that go without saying, like "Today I am a day older than I was yesterday". The strongest statements are those which just can't be true, are unconditionally false, self-contradictory statements like "tomorrow it will rain, and tomorrow it will not rain".

A statement of type p∧qp \wedge qp∧q is stronger than ppp, because for it to be true, both ppp and qqq have to be true. Thus p∧q≤pp \wedge q \leq pp∧q≤p. If you agree with that, then you have to agree with that in the case q=¬pq = \neg pq=¬p, where p∧¬pp \wedge \neg pp∧¬p is unconditional falsity, is equivalent to ⊥\bot⊥. Thus ⊥≤p\bot \leq p⊥≤p for any statement ppp.

Todd Trimble (Aug 20 2024 at 13:08):

This makes me think the fixed-point theorem is very much related to cardinality.

Yes. For example, Lawvere's fixed point theorem subsumes Cantor's theorem, which says that for any set XXX, the power set 2X2^X2X has greater cardinality than XXX. Because if 2X2^X2X has the same cardinality as XXX, then there exists a bijection X≅2XX \cong 2^XX≅2X, which in turn means that there exists a surjection X→2XX \to 2^XX→2X, which by Lawvere's theorem means that every endofunction 2→22 \to 22→2 has a fixed point. But this is absurd, since the transposition on the set 2={0,1}2 = \{0, 1\}2={0,1} that swaps 000 and 111 obviously has no fixed point!

There is nothing particularly special about 222 here. For any set AAA, as soon as you know how to construct an endofunction on AAA that has no fixed point, the same reasoning shows that ABA^BAB and BBB must have different cardinalities, for any set BBB. This applies in particular to A=NA = \mathbb{N}A=N, because as you pointed out in Lemma 1, the successor endofunction on N\mathbb{N}N has no fixed point.

So your lemma 5, for example, follows fairly readily from Lawvere's theorem if we prove that any set AAA such that ∣A∣>1|A| > 1∣A∣>1 has a fixed-point-free endofunction on it. One way of constructing such an endofunction starts off by well-ordering AAA, and then considers the function that takes all but the greatest element of AAA (if there is one) to its successor (exercise: what is that?), and takes the greatest element (if there is one) back to the first element.

Part of my point is that you don't need to prove lemmas 2 through 5 "off to the side", away from Lawvere's theorem. They follow from Lawvere's theorem.

Ryan Wisnesky (Aug 27 2024 at 00:44):

To turn the STLC into a CCC, as shown in e.g. Awodeys textbook, you consider the objects to be types of the STLC and a morphism x to y to be a term of type y with a free variable of type x; the STLC equations such as beta and eta allow one to establish the required universal properties of products, exponentials, etc. To turn an equational theory in a CC you do the same thing, but since you don't have lambda anymore you end up with a cartesian category, i.e., one with all products. These are also called 'multi-sorted equational theories' and 'algebraic theories' in the literature. Anyway, you can form a category of theories (equational, Horn, etc), where the morphisms from theory A to theory B (on a possible different signature) are 'theory morphisms': mappings from one to signature to another that preserve provability: if F : A -> B is a theory morphism, then if p = q is provable in A, F(p)=F(q) should be provable in B. Typically theory morphisms are expressed as mappings from the symbols in A to symbols in B (or to open terms in "B". Think mapping "+" to "*" for example).
