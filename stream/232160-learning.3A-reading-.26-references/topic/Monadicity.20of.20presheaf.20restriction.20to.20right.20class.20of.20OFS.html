Topic: Monadicity of presheaf restriction to right class of OFS

Amar Hadzihasanovic (Jun 08 2025 at 15:19):

The following fact is quite easy to prove: if CCC is a category with a strict factorisation system (L,R)(\mathcal{L}, \mathcal{R})(L,R), and CRC_\mathcal{R}CR​ is the wide subcategory on R\mathcal{R}R-morphisms, then the presheaf restriction functor PSh(C)→PSh(CR)\mathrm{PSh}(C) \to \mathrm{PSh}(C_\mathcal{R})PSh(C)→PSh(CR​) is monadic.
One can explicitly describe the monad on PSh(CR)\mathrm{PSh}(C_\mathcal{R})PSh(CR​) as sending a presheaf XXX to the presheaf whose ccc-elements are pairs (ℓ ⁣:c→d,x∈X(d))(\ell\colon c \to d, x \in X(d))(ℓ:c→d,x∈X(d)) where ℓ\ellℓ is a L\mathcal{L}L-morphism; one uses the unique factorisation to define the action of R\mathcal{R}R-morphisms on these elements.
Does anyone know an explicit reference for this fact?

Nathanael Arkor (Jun 08 2025 at 17:19):

This is an instance of the fact that restriction along [[dominant functors]] (and, in particular, identity-on-object functors) is monadic. E.g. a reference is Example A4.2.7(b) of the Elephant.
