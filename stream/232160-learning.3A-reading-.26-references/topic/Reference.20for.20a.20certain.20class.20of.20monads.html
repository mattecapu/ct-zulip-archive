Topic: Reference for a certain class of monads

Regard the monoid of natural numbers as a bicategory with a single object, and only identity 2-cells.

Then, for every monoidal category VVV, you can form the bicategory of pseudofunctors N→V\mathbb N \to VN→V, where an object is an object AAA of VVV, a 1-cell (X,ϕ)(X,\phi)(X,ϕ) is a morphism ϕ:A⊗X→X⊗B\phi : A\otimes X \to X\otimes Bϕ:A⊗X→X⊗B in VVV, and a 2-cell α:(X,ϕ)→(Y,ψ)\alpha : (X,\phi) \to (Y,\psi)α:(X,ϕ)→(Y,ψ) is a certain morphism in VVV, α:X→Y\alpha : X\to Yα:X→Y, plus the obvious compatibility with ϕ,ψ\phi,\psiϕ,ψ.
