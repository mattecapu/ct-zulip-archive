Puzzle. Let X=RX = \mathbb{R}X=R and for each open set U⊆RU \subseteq \mathbb{R}U⊆R take FUF UFU to be the set of continuous real-valued functions on UUU. Show that with the usual concept of restriction of functions, FFF is a presheaf and in fact a sheaf.

David Egolf (Mar 27 2024 at 17:39):

To show that FFF is a presheaf, we need to show that it is a functor F:O(R)op→SetF: \mathcal{O}(\mathbb{R})^{\mathrm{op}} \to \mathsf{Set}F:O(R)op→Set. Now, for each open U⊆RU \subseteq \RU⊆R, we have that FUFUFU is the set of continuous-real valued functions on UUU. To talk about "continuous" functions means that there needs to be some topology on UUU. I think it's reasonable to assume that UUU is equipped with the subspace topology it inherits from R\mathbb{R}R.

I'm a bit intrigued by the fact that we then have FU=Top(U,R)FU = \mathsf{Top}(U, \mathbb{R})FU=Top(U,R), where UUU is equipped with the subspace topology. This leads me to consider the functor Top(−,R):Topop→Set\mathsf{Top}(-,\mathbb{R}): \mathsf{Top}^{\mathrm{op}} \to \mathsf{Set}Top(−,R):Topop→Set.

David Egolf (Mar 27 2024 at 17:43):

I'd now like dream up some functor G:O(R)op→TopopG: \mathcal{O}(\mathbb{R})^{\mathrm{op}} \to \mathsf{Top}^{\mathrm{op}}G:O(R)op→Topop, so that we can express FFF as F=Top(−,R)∘GF = \mathsf{Top}(-, \mathbb{R}) \circ GF=Top(−,R)∘G.

The first idea that comes to mind is to let GGG act as follows:

  • it sends each open subset of R\mathbb{R}R to that same subset, viewed as a topological space equipped with the subspace topology it inherits from R\mathbb{R}R
  • it sends each morphism U→VU \to VU→V to the morphism from UUU to VVV corresponding to the inclusion function from VVV to UUU
