John Baez (Apr 06 2021 at 15:00):

You can estimate the growth rate of the Catalan numbers by applying the Cauchy-Hadamard theorem to the function

c(z)=1−1−4z2z \displaystyle{ c(z) = \frac{1 - \sqrt{1 - 4z}}{2z} } c(z)=2z1−1−4z​​

But it's much harder, maybe impossible to get an exact formula this way. The Fibonacci generating function f(z)f(z)f(z) is a rational function, so you can estimate the growth rate of the Fibonacci numbers by using Cauchy-Hadamard, which only cares about the pole closest to the origin, and get an exact formula by keeping track of both poles. But c(z)c(z)c(z) has a branch cut so it's much more complicated.
