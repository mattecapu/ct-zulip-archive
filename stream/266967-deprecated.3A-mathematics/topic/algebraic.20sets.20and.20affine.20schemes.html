John Baez (Apr 18 2023 at 02:24):

Jean-Baptiste Vienney said:

Do you have an idea if there is a characterization of the affine schemes which are the set of zeroes of a polynomial?

This is a slightly strange question because an affine scheme is not a set. Also, people are much more interested in collections of polynomials, not a single polynomial. But maybe you are trying to ask a hard question.

Say we have a field kkk and a finite collection of polynomials P1,…,Pm∈k[x1,…,xn]P_1, \dots, P_m \in k[x_1, \dots, x_n]P1​,…,Pm​∈k[x1​,…,xn​]. We can do two different things.

First, we can look at the set X⊆knX \subseteq k^nX⊆kn where all these polynomials vanish. This sort of set is called an algebraic set. Without loss of generality we can take this collection to be finite, say P1,…,PmP_1, \dots, P_mP1​,…,Pm​.

Second, we can form the commutative ring k[x1,…,xn]/Jk[x_1, \dots, x_n]/Jk[x1​,…,xn​]/J where JJJ is the ideal generated by P1,…,PmP_1, \dots, P_mP1​,…,Pm​. Since affine schemes correspond to commutative rings, this is an affine scheme.

So, there are two questions we could ask here:

1) which subsets of knk^nkn are algebraic sets?

2) which affine schemes are of the form k[x1,…,xn]/Jk[x_1, \dots, x_n]/Jk[x1​,…,xn​]/J where JJJ is the ideal generated by P1,…,PmP_1, \dots, P_mP1​,…,Pm​?

Question 2) has an easy answer: they are just the affine schemes whose corresponding commutative rings are finitely presented kkk-algebras. And interestingly, these are the same as finitely generated kkk-algebras.

For any sort of gadget with operations obeying equations we can talk about a "presentation" in terms of generators and relations. Examples include monoids, groups, rings, vector spaces, and what I'm talking about now: associative unital algebras over a field. We say a gadget is "finitely presented" if it has a presentation with finitely many generators and finitely many relations.

Okay, the interesting thing about R3\mathbb{R}^3R3 is that the two equations

x=0,y=0x = 0, \qquad y = 0x=0,y=0

are equivalent to just one equation:

x2+y2=0 x^2 + y^2 = 0 x2+y2=0

In general, we can use this trick to take any finite number of polynomial equations in nnn real variables and write them as just one equation!
