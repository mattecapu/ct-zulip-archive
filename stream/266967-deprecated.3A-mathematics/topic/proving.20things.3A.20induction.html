David Egolf (Apr 03 2023 at 15:34):

In the spirit of some recent threads, I have been attempting to write a proof for an exercise in "A Mathematical Introduction to Logic" by Enderton, and I was hoping for some feedback on the proof structure I would like to use. I'm also hoping this thread can be of interest to other people hoping to learn how to prove things.

In this exercise, I find myself wanting to use a form of induction a bit more general than what I've used before, and I'm not sure if it's valid.

The exercise in question states:
"Let α be a wff; let c be the number of places at which binary connective symbols (∧, ∨, →, ↔) occur in α; let s be the number of places at which sentence symbols occur in α. (For example, if α is (A → (¬ A)) then c = 1 and s = 2.) Show by using the induction principle that s = c + 1."

Here a "wff" refers to a "well-formed formula", which is created in one of the following ways:

  • Every "sentence symbol" (namely, one of: A1, A2, A3,....) is a wff
  • If α\alphaα and β\betaβ are wffs, then so are (¬ α), (α ∧ β), (α ∨ β), (α → β),
    and (α ↔ β). We call these ways of making new wffs from old ones "formula-building operations".
    Every wff is constructed from sentence symbols by application of a finite number of formula-building operations.

Now, the usual form of an induction argument I'm familiar with goes roughly like this:

  • Form a statement that depends on a natural number, P(n)P(n)P(n)
  • Show that the statement holds for the smallest natural number of interest sss
  • Show that if the statement holds for nnn, it holds for n+1n+1n+1
  • Conclude that P(n)P(n)P(n) holds for all natural numbers greater than or equal to sss

However, the induction I'd like to do in this case has a bit of a different structure:

  • Show that all wffs formed from 0 formula-building operations satisfy s = c+1.
  • Show that applying any formula-building operation to wffs satisfying s = c+1 yields a wff that satisfies s = c+1
  • Conclude that all wffs satisfy s = c+1, since all wffs are made from a finite number of formula-building operations

The second step described here (a sort of inductive step) is straightforward to carry out. What I'm more concerned with here is the basic structure of the attempted proof involved. It feels like I'm trying to generalize the idea from normal induction where we make use of repeated applications of "add one" to generate all natural numbers of interest. Is there a more general form of induction where we talk about using a collection of operations which together generate all the objects of interest, from a set of "base case" objects?

That is, is the following argument structure valid?

  • We have a statement P(x)P(x)P(x) which depends on x∈Xx \in Xx∈X. We want to show P(x)P(x)P(x) is true for all x∈Xx \in Xx∈X.
  • Begin by showing P(x)P(x)P(x) is true for all x∈Yx \in Yx∈Y, with Y⊆XY \subseteq XY⊆X.
  • Introduce a number of operations b1,b2,…b_1, b_2, \dotsb1​,b2​,… of potentially varying arity NiN_iNi​ that "generate" XXX from YYY. That is, given any element xxx of XXX, by applying our operations a finite number of times starting with elements in YYY, we can produce the specified element xxx
  • Show that, for each iii, the output of bi:XNi→Xb_i: X^{N_i} \to Xbi​:XNi​→X always satisfies PPP, provided that each of the NiN_iNi​ input elements satisfy PPP
  • Conclude that P(x)P(x)P(x) is true for each x∈Xx \in Xx∈X

Any thoughts around this topic are welcome!

Ryan Wisnesky (Apr 03 2023 at 15:47):

The structure of a proof based on induction over the natural numbers should always be the same: first, you write a predicate over Nat, say P, and then show that P(0) holds, and then that P(n) implies P(n+1) (or P(m) for all m <= n implies P(n), for strong induction). Is it possible you are trying to do induction over the syntax of wffs, where you first write a predicate over wffs, say, Q, and then show that Q holds of all atomic propositions, and then that Q(x*y) holds assuming Q(x) and Q(y), hold for each connective *. It's also possible to do induction over proofs of wffs, any many things besides. In any case I'd encourage you to write the inductive hypothesis explicitly.

Mike Shulman (Apr 03 2023 at 20:32):

Well-founded partial orders are one way to be precise about structural induction, but not the only way. For instance, your sketch:

  • We have a statement P(x)P(x)P(x) which depends on x∈Xx \in Xx∈X. We want to show P(x)P(x)P(x) is true for all x∈Xx \in Xx∈X.
  • Begin by showing P(x)P(x)P(x) is true for all x∈Yx \in Yx∈Y, with Y⊆XY \subseteq XY⊆X.
  • Introduce a number of operations b1,b2,…b_1, b_2, \dotsb1​,b2​,… of potentially varying arity NiN_iNi​ that "generate" XXX from YYY. That is, given any element xxx of XXX, by applying our operations a finite number of times starting with elements in YYY, we can produce the specified element xxx
  • Show that, for each iii, the output of bi:XNi→Xb_i: X^{N_i} \to Xbi​:XNi​→X always satisfies PPP, provided that each of the NiN_iNi​ input elements satisfy PPP
  • Conclude that P(x)P(x)P(x) is true for each x∈Xx \in Xx∈X

can be justifed directly without talking about well-foundedness explicitly. Define a subset ZZZ of XXX to be "bbb-closed" if whenever the inputs of some bib_ibi​ belong to ZZZ, so does the output. Your assumption that YYY generates XXX under bbb means that if a bbb-closed subset contains YYY, then it must be all of XXX. (If you like, this is by ordinary natural-number induction on the finite number of bbb-operations you have to apply to get from YYY to some element x∈Xx\in Xx∈X.) Your two proofs show that the set {x∈X∣P(x)}\{x \in X \mid P(x)\}{x∈X∣P(x)} contains YYY and is bbb-closed; therefore it is all of XXX.

Mike Shulman (Apr 03 2023 at 20:33):

Another way to justify structural induction is to talk about general "inductive definitions", which categorically are [[initial]] [[algebras for an endofunctor]]. In the case of logical formulas (I dislike the term "wff" because no one ever wants to talk about an "ill-formed" formula), we are talking about the endofunctor of Set\rm SetSet defined by
F(x)=S+X+(X×X)+(X×X)+(X×X)+(X×X) F(x) = S + X + (X\times X) + (X\times X) + (X\times X) + (X\times X) F(x)=S+X+(X×X)+(X×X)+(X×X)+(X×X)
where SSS is the set of sentence symbols. An FFF-algebra is a set XXX with a function F(X)→XF(X) \to XF(X)→X, which means that we have an interpretation of every sentence symbol as an element of XXX, together with one unary operation X→XX\to XX→X corresponding to negation, and four binary operations X×X→XX\times X \to XX×X→X corresponding to disjunction, conjunction, implication, and equivalence. The set WWW of formulas is the initial such FFF-algebra, which means that its elements are "freely generated" by applying these operations formally. This is a categorical way to express that "Every wff is constructed from sentence symbols by application of a finite number of formula-building operations."

Given this definition, we can justify structural induction as follows. Suppose P(x)P(x)P(x) is some property of formulas, and consider U={x∈W∣P(x)}U = \{ x\in W \mid P(x)\}U={x∈W∣P(x)}. The "base case" and "inductive steps" show that UUU is itself an FFF-algebra such that the inclusion U↪WU \hookrightarrow WU↪W is an FFF-algebra homomorphism. But WWW is the initial FFF-algebra, so there is also an FFF-algebra homomorphism W→UW\to UW→U, and uniqueness means that the composite W→U↪WW\to U \hookrightarrow WW→U↪W is the identity. Therefore, U=WU = WU=W, i.e. P(x)P(x)P(x) holds for all x∈Wx\in Wx∈W.

Here's the principle of strong induction as stated in "An Infinite Descent into Pure Mathematics":
strong induction

We set p(n)p(n)p(n) to be: "A wff constructed by application of nnn formula-building operations satisfies s=c+1s=c+1s=c+1."
p(0)p(0)p(0) is: A wff constructed by application of 0 formula-building operations satisfies s=c+1s=c+1s=c+1. This is true because s=1s=1s=1 and c=0c=0c=0 for our sentence symbols, which are the only wffs formed by applying zero formula-building operations.

For arbitrary n≥0n \geq 0n≥0, we now assume that p(k)p(k)p(k) is true for all 0≤k≤n0 \leq k \leq n0≤k≤n.
We want to show that p(n+1)p(n+1)p(n+1) is true under this assumption.
p(n+1)p(n+1)p(n+1) is: A wff constructed by application of n+1n+1n+1 formula-building operations satisfies s=c+1s=c+1s=c+1.
Let γ\gammaγ be a wff constructed by application of n+1n+1n+1 formula-building operations.
Then γ\gammaγ is of one of the following forms:
(¬ α), (α ∧ β), (α ∨ β), (α → β)
where α\alphaα and β\betaβ are wffs.
In each case, γ\gammaγ is constructed from wffs constructed by application of nnn or fewer formula-building operations.
In the first case, γ=(¬α)\gamma = (\neg \alpha)γ=(¬α). We know αs=αc+1\alpha_s = \alpha_c + 1αs​=αc​+1 by our assumption.
(Here αs\alpha_sαs​ denotes sss for the wff α\alphaα, and αc\alpha_cαc​ denotes ccc for α\alphaα).
So, γs=αs\gamma_s = \alpha_sγs​=αs​ and γc=αc\gamma_c = \alpha_cγc​=αc​ and γs=γc+1\gamma_s = \gamma_c + 1γs​=γc​+1.
In the second case γ=α∧β\gamma = \alpha \wedge \betaγ=α∧β, we know αs=αc+1\alpha_s = \alpha_c + 1αs​=αc​+1 and βs=βc+1\beta_s = \beta_c + 1βs​=βc​+1 by our assumption.

We conclude p(n+1)p(n+1)p(n+1) is true for arbitrary n≥0n \geq 0n≥0, assuming that p(k)p(k)p(k) is true for all 0≤k≤n0 \leq k \leq n0≤k≤n.
