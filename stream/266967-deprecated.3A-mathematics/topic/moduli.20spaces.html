John Baez (Sep 23 2023 at 20:11):

However, to apply a given permutation I think we need to put an ordering on the vertices of an acute triangle determined by an element of T.

Don't think about T at first, think about a triangle in the plane with vertices labeled 1, 2, 3. Permutations of the set {1,2,3}\{1,2,3\}{1,2,3} act on any such triangle to give a new labeled triangle - the same triangle, just with a different labeling. There's nothing about an ordering involved here, though of course the set 1, 2, 3 is ordered if you want it to be.

T is the set of isomorphism classes of labeled acute triangles in the plane, where 'isomorphism' means that two labeled triangles are related by a similarity (rotation, reflection and/or rescaling) in a way that preserves the labelings, mapping vertex 1 of the first triangle to vertex 1 of the second one, etc.

Since S3S_3S3​ acts on the set of labeled triangles in the plane it acts on the set of isomorphism classes of these.

That's right, though most mathematicians would not use the word "extend" here. If I have function or group action on some set SSS and some larger set S′S'S′ with S′S'S′, I might want to "extend" my function or group action from SSS to S′S'S′. But here we have a set SSS and a quotient set S′S'S′, i.e. a set of equivalence classes of elements of S′S'S′. So we're more likely to ask if the function or action "descends" from SSS to S′S'S′.

(We think of elements of S′S'S′ as being below those of SSS, like shadows of objects: different objects can have the same-looking shadow.)

Some hardcore category theorists might say "extend" even in this case, since the two cases have this in common: we have a map f:S→S′f: S \to S'f:S→S′. Category theorists might ask if a group action "extends along fff". But normal mathematicians would be put off by this when fff is not an injection.
