Verity Scheel (Apr 04 2020 at 00:44):

Can we prove it? Let a and b be two distinct almost-identities in a commutative (sub)monoid. Then either a absorbs b (a > b), in which case ab = a, or a is absorbed by b (a < b), in which case ab = b (trichotomy). Due to commutativity, if a absorbs b, then b is absorbed by a, and vice-versa, so it is anti-symmetric or something. Just need transitivity … a < b < c so ab = b and bc = c so ac = a(bc) = (ab)c = bc = c. Looks legit to me :)
