Topic: what is this construction called?

Paolo Capriotti (Jan 15 2021 at 09:11):

If MMM is a monoid and XXX is a right MMM-set, the set of functions X→MX \to MX→M has a monoid structure, defined by (fg)(x)=f(x)g(xf(x))(fg)(x) = f(x) g(x f(x))(fg)(x)=f(x)g(xf(x)). Does this have a name? Can it be expressed more abstractly?

There is also a group version of this. If GGG is a group, and XXX is now a GGG-set, we can take the subset of those f ⁣:X→Gf\colon X \to Gf:X→G such that the map x↦xf(x)x \mapsto x f(x)x↦xf(x) is invertible. This is then a subgroup of the above monoid.
