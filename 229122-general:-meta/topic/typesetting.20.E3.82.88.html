Alastair Grant-Stuart (Nov 23 2020 at 09:15):

If you're on Linux (or more specifically, if you use the Compose key) you can set up a custom Compose key sequence, e.g. by making a file named .XCompose in your home folder with the following content:

include "%L"    # import default rules from system Compose file
<Multi_key> <y> <o> : "よ" U+3088    # YO kana

Then you can type よ with key combination <Compose key> + <y> + <o>.
